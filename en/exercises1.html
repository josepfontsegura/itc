Exercises: 1. Sequences of random variables

Exercise 1.1. Knowing that the probability that the source outputs the sequence $210$ is $\frac{1}{36}$, what is the probability that the source outputs the sequence $02$?

(a) $\frac{1}{12}$
(b) $\frac{1}{6}$
(c) Both (a) and (b) are compatible answers.

Show solution

The correct answer is (c) ★

The statement implies that $P_U(2)=\frac 12$ and therefore that $P_{U_1U_2U_3}(210)=\frac 12P_U(1)P_U(0)=\frac{1}{36}$. Together with $P_U(0)+P_U(1)=\frac 12$, we find that the distribution is either $\bigl\{ \frac 16,\frac 13,\frac 12 \bigr\}$ or $\bigl\{ \frac 13,\frac 16,\frac 12 \bigr\}$. The question asks for $P_U(0)P_U(2)$, which can be wither $\frac{1}{12}$ or $\frac{1}{6}$.

Exercise 1.2. Which is the probability that the source does not output the all-$2$ sequence?

(a) $\frac{1}{2^n}$
(b) $\frac{1}{4^n}$
(c) $1-2^{-n}$

Show solution

The correct answer is (c) ★

Since the probability that the source outputs the all-$2$ sequence is $P_{U_1\cdots U_n}(2\cdots 2)=P_U(2)^n=2^{-n}$, the complementary event happens with probability $1-2^{-n}$.

Exercise 1.3. Which of the following pairs of sequences do not have the same probability?

(a) $0012$ and $0201$
(b) $1111$ and $2222$
(c) $1212$ and $2121$

Show solution

The correct answer is (b) ★

As a discrete memoryless source, the probability of a sequence only depends on the number of occurrences of each symbol.

Exercise 1.4. Which is the probability that the output sequence is $abb$ conditioned on the input sequence being $cba$?

(a) $0$
(b) $\frac 13$
(c) None of the others.

Show solution

The correct answer is (b) ★

The probability is given by $$ P_{Y|X}(a|c)P_{Y|X}(b|b)P_{Y|X}(b|a) = \frac 12 \cdot 1 \cdot \frac 23 = \frac 13. $$

Exercise 1.5. Which is the probability that, knowing that $(X_1,X_2,X_3)=bbb$, $(Y_1,Y_2,Y_3)$ is $aaa$?

(a) $1$
(b) $0$
(c) $\frac 18$

Show solution

The correct answer is (b) ★

Conditioned on $(X_1,X_2,X_3)=bbb$, the probability that the output sequence is $aaa$ is $\mathbb{P}[(Y_1,Y_2,Y_3)=aaa|(X_1,X_2,X_3)=bbb] = P_{Y_1Y_2Y_3|X_1X_2X_3}(aaa|bbb)$. Since the channel is memoryless, the conditional distribution is given by $P_{Y_1Y_2Y_3|X_1X_2X_3}(aaa|bbb)=P_{Y|X}(a|b)^3=0^3=0$.

Exercise 1.6. Which is the probability that $(Y_1,Y_2,Y_3)=bbb$?

(a) $\frac{59}{648}$
(b) $\frac{307}{648}$
(c) $\frac{35}{648}$

Show solution

The correct answer is (b) ★

By the total probability law, we can calculate the probability that $(Y_1,Y_2,Y_3)$ is $bbb$ as $\sum_{(x_1,x_2,x_3)\in\{a,b,c\}^3} \mathbb{P}[(Y_1,Y_2,Y_3)=bbb|(X_1,X_2,X_3)=(x_1,x_2,x_3)]P_{X_1X_2X_3}(x_1,x_2,x_3)$. We know $(X_1,X_2,X_3)$ only takes three possible values with non-zero probability, so we only have to calculate the following probabilities $$ \mathbb{P}[(Y_1,Y_2,Y_3)=bbb|(X_1,X_2,X_3)=aaa] = \tfrac{8}{27}\\ \mathbb{P}[(Y_1,Y_2,Y_3)=bbb|(X_1,X_2,X_3)=bbb] = 1\\ \mathbb{P}[(Y_1,Y_2,Y_3)=bbb|(X_1,X_2,X_3)=ccc] = \tfrac 18. $$ For example, to find the first one we used that $ P_{Y_1Y_2Y_3|X_1X_2X_3}(bbb|aaa)=P_{Y|X}(b|a)^3$. Plugging the numbers in the summation and noting that $P_{X_1X_2X_3}(bbb)=P_{X_1X_2X_3}(ccc)=P_{X_1X_2X_3}(aaa)=\frac 13$, we obtain the result.

Exercise 1.7. The distribution of $(U_1,U_2)$, in lexicographic order, is given by

(a) $\bigl\{ \frac{1}{4},\frac{1}{12},\frac{1}{6},\frac{1}{12},\frac{1}{36},\frac{1}{18},\frac{1}{6},\frac{1}{18},\frac{1}{9} \bigr\}$
(b) $\bigl\{ \frac 19, \frac{1}{18}, \frac{1}{6}, \frac{1}{18},\frac{1}{36} , \frac{1}{12}, \frac{1}{6}, \frac{1}{12}, \frac{1}{4}\bigr\}$
(c) None of the others.

Show solution

The correct answer is (a) ★

Together with $P_U(0)+P_U(1)+P_U(2)=1$, we find that $P_U(0)=\frac 12$, $P_U(1)=\frac 16$ and $P_U(2)=\frac 13$. Since symbols are i.i.d., the distribution of $(U_1,U_2)$ satisfies $P_{U_1U_2}(u_1,u_2)=P_U(u_1)P_U(u_2)$, for each $(u_1,u_2)\in\{0,1,2\}^2=\{00,01,02,10,11,12,20,21,22\}$, leading to the result.

Exercise 1.8. Which is the probability that the source outputs a sequence containing at least one symbol $0$?

(a) $\frac 34$
(b) $\frac 12$
(c) $\frac 49$

Show solution

The correct answer is (a) ★

The event ``$(U_1,U_2)$ has exactly one symbol $0$'' is described by the set of outut sequences containing exactly one symbol $0$, that is ${\cal A}=\{00,01,02,10,20\}$. Therefore, $\mathbb{P}[\text{$(U_1,U_2)$ has exactly one symbol $0$}]=\mathbb{P}[(U_1,U_2)\in{\cal A}]=\sum_{(u_1,u_2)\in{\cal A}}P_{U_1U_2}(u_1,u_2)$. Using the distribution in question 1, we obtain the result.

Exercise 1.9. Which is the probability that the source outputs the all $0$ sequence?

(a) $\frac{1}{3^n}$
(b) $\frac{1}{2^n}$
(c) $\frac{1}{6^n}$

Show solution

The correct answer is (b) ★

As a sequence of i.i.d. random variables, we have that $\mathbb{P}[(U_1,\ldots,U_n)=0\cdots 0]=P_{U_1\cdots U_n}(0\cdots 0) = \prod_{i=1}^n P_U(0) = \frac{1}{2^n}.$

Exercise 1.10. Which of the following corresponds to the probability distribution of $Y$?

(a) $\bigl\{ \frac 13P_X(a)+P_X(b)+\frac 12 P_X(c) , \frac 23 P_X(a) + \frac 12 P_X(c) \bigr\}$
(b) $\bigl\{ \frac 13P_X(a)+\frac 12 P_X(c) , \frac 23 P_X(a) + P_X(b) + \frac 12 P_X(c) \bigr\}$
(c) None of the others

Show solution

The correct answer is (b) ★

The probability distribution of $Y$ is given by $\{P_Y(a),P_Y(b)\}$. By the total probability law, $P_Y(y) = \sum_{x\in\{a,b,c\}}P_{Y|X}(y|x)P_X(x)$. For example, $P_Y(a) = P_{Y|X}(a|a)P_X(a) + P_{Y|X}(a|b)P_X(b) + P_{Y|X}(a|c)P_X(c)$. Substituting the numbers we obtain the result.

Exercise 1.11. Which is the probability that $(Y_1,Y_2,Y_3)=aaa$?

(a) $\frac{59}{648}$
(b) $\frac{59}{324}$
(c) $\frac{35}{648}$

Show solution

The correct answer is (c) ★

By the total probability law, we can calculate the probability that $(Y_1,Y_2,Y_3)$ is $aaa$ as $\sum_{(x_1,x_2,x_3)\in\{a,b,c\}^3} \mathbb{P}[(Y_1,Y_2,Y_3)=aaa|(X_1,X_2,X_3)=(x_1,x_2,x_3)]P_{X_1X_2X_3}(x_1,x_2,x_3)$. We know $(X_1,X_2,X_3)$ only takes three possible values with non-zero probability, so we only have to calculate the following probabilities $$ \mathbb{P}[(Y_1,Y_2,Y_3)=aaa|(X_1,X_2,X_3)=aaa] = \tfrac{1}{27}\\ \mathbb{P}[(Y_1,Y_2,Y_3)=aaa|(X_1,X_2,X_3)=bbb] = 0\\ \mathbb{P}[(Y_1,Y_2,Y_3)=aaa|(X_1,X_2,X_3)=ccc] = \tfrac 18. $$ For example, to find the first one we used that $ P_{Y_1Y_2Y_3|X_1X_2X_3}(aaa|aaa)=P_{Y|X}(a|a)^3$. Plugging the numbers in the summation and noting that $P_{X_1X_2X_3}(bbb)=P_{X_1X_2X_3}(ccc)=P_{X_1X_2X_3}(aaa)=\frac 13$, we obtain the result.

Exercise 1.12. A discrete memoryless source produces i.i.d. symbols from the alphabet $\{a,b\}$ with $P(a)=\tfrac{1}{3}$. What is the probability of the sequence $abba$?

(a) $\tfrac{4}{81}$
(b) $\tfrac{2}{27}$
(c) $\tfrac{4}{27}$

Show solution

The correct answer is (a) ★

Since the source is memoryless, $P(abba) = P(a)P(b)P(b)P(a) = \tfrac{1}{3}\cdot\tfrac{2}{3}\cdot\tfrac{2}{3}\cdot\tfrac{1}{3} = \tfrac{4}{81}$.

Exercise 1.13. Let $X$ and $Y$ be two independent random variables with $X\in\{0,1\}$ equiprobable and $Y\in\{0,1,2\}$ equiprobable. What is $\mathbb{E}[XY]$?

(a) $\tfrac{1}{2}$
(b) $1$
(c) $\tfrac{1}{3}$

Show solution

The correct answer is (a) ★

By independence, $\mathbb{E}[XY]=\mathbb{E}[X]\mathbb{E}[Y]=\tfrac{1}{2}\cdot 1 = \tfrac{1}{2}$.

Exercise 1.14. A ternary source emits symbols from $\{0,1,2\}$ with probabilities $\{\tfrac{1}{2},\tfrac{1}{4},\tfrac{1}{4}\}$. For a sequence of length $n=2$, how many distinct sequences have probability $\tfrac{1}{8}$?

(a) $2$
(b) $4$
(c) $1$

Show solution

The correct answer is (b) ★

A sequence has probability $\tfrac{1}{8}$ when it consists of one symbol with probability $\tfrac{1}{2}$ and one with probability $\tfrac{1}{4}$. The possible sequences are $01$, $02$, $10$, $20$, giving $4$ sequences.

Exercise 1.15. A discrete memoryless source emits i.i.d. symbols from $\{a,b,c\}$ with $P(a)=\tfrac{1}{2}$, $P(b)=\tfrac{1}{3}$, $P(c)=\tfrac{1}{6}$. What is the probability that a sequence of length $2$ contains no $a$?

(a) $\tfrac{1}{4}$
(b) $\tfrac{1}{2}$
(c) $\tfrac{2}{9}$

Show solution

The correct answer is (a) ★

The probability of no $a$ in a single symbol is $1-\tfrac{1}{2}=\tfrac{1}{2}$. For two independent symbols, the probability is $\bigl(\tfrac{1}{2}\bigr)^2=\tfrac{1}{4}$.

Exercise 1.16. Consider a binary symmetric channel with crossover probability $p=\tfrac{1}{4}$. If the input is $X=0$, what is the probability that the output sequence $(Y_1,Y_2)$ equals $(0,1)$ when two independent uses of the channel are made?

(a) $\tfrac{3}{16}$
(b) $\tfrac{1}{4}$
(c) $\tfrac{1}{16}$

Show solution

The correct answer is (a) ★

Since the channel is memoryless, $P(Y_1=0,Y_2=1|X_1=0,X_2=0)=P(Y=0|X=0)P(Y=1|X=0)=(1-p)p=\tfrac{3}{4}\cdot\tfrac{1}{4}=\tfrac{3}{16}$.

Exercise 1.17. A fair coin is flipped $n$ times independently. Let $S_n$ be the number of heads. What is $\mathbb{E}[S_n/n]$?

(a) $\tfrac{1}{2}$
(b) $\tfrac{1}{n}$
(c) $1$

Show solution

The correct answer is (a) ★

Since $\mathbb{E}[S_n]=\tfrac{n}{2}$ for a fair coin, we have $\mathbb{E}[S_n/n]=\tfrac{1}{2}$.

Exercise 1.18. Let $(X_1,X_2,X_3)$ be i.i.d. with $P(X_i=0)=\tfrac{1}{3}$ and $P(X_i=1)=\tfrac{2}{3}$. What is the most probable sequence?

(a) $111$
(b) $000$
(c) $110$

Show solution

The correct answer is (a) ★

Since $P(1)>P(0)$, the most probable sequence is the all-ones sequence, with probability $\bigl(\tfrac{2}{3}\bigr)^3=\tfrac{8}{27}$.

Exercise 1.19. A discrete memoryless channel has input $X\in\{0,1\}$ and output $Y=X$. If $P_X(0)=\tfrac{3}{4}$, what is $P_Y(1)$?

(a) $\tfrac{1}{4}$
(b) $\tfrac{3}{4}$
(c) $\tfrac{1}{2}$

Show solution

The correct answer is (a) ★

Since $Y=X$ deterministically, $P_Y(1)=P_X(1)=1-\tfrac{3}{4}=\tfrac{1}{4}$.

Exercise 1.20. Two random variables $X\in\{0,1\}$ and $Y\in\{0,1\}$ satisfy $P_{XY}(0,0)=P_{XY}(1,1)=\tfrac{1}{2}$. Are $X$ and $Y$ independent?

(a) No, because $P_{XY}(0,0)\neq P_X(0)P_Y(0)$
(b) Yes, because $P_X(0)=P_Y(0)=\tfrac{1}{2}$
(c) It depends on the marginals

Show solution

The correct answer is (a) ★

We have $P_X(0)=P_Y(0)=\tfrac{1}{2}$, so $P_X(0)P_Y(0)=\tfrac{1}{4}\neq\tfrac{1}{2}=P_{XY}(0,0)$. Therefore $X$ and $Y$ are not independent.

Exercise 1.21. Let $X$ be a random variable taking values in $\{1,2,3,4\}$ with equal probability. What is $\text{var}(X)$?

(a) $\tfrac{5}{4}$
(b) $\tfrac{5}{3}$
(c) $\tfrac{3}{2}$

Show solution

The correct answer is (a) ★

We have $\mathbb{E}[X]=\tfrac{10}{4}=\tfrac{5}{2}$ and $\mathbb{E}[X^2]=\tfrac{1+4+9+16}{4}=\tfrac{30}{4}=\tfrac{15}{2}$. Therefore $\text{var}(X)=\tfrac{15}{2}-\tfrac{25}{4}=\tfrac{5}{4}$.

Exercise 1.22. Given an i.i.d. binary source with $P(0)=\tfrac{1}{4}$, find the probability of the sequence $0110$.

(a) $\tfrac{9}{256}$
(b) $\tfrac{3}{64}$
(c) $\tfrac{1}{16}$

Show solution

The correct answer is (a) ★

Since the source is i.i.d., $P(0110) = P(0)P(1)P(1)P(0) = \tfrac{1}{4}\cdot\tfrac{3}{4}\cdot\tfrac{3}{4}\cdot\tfrac{1}{4} = \tfrac{9}{256}$.

Exercise 1.23. For a memoryless channel with transition probabilities $P(Y{=}0|X{=}0)=0.9$, $P(Y{=}0|X{=}1)=0.2$, what is $P(Y{=}0|X{=}1)$?

(a) $0.2$
(b) $0.8$
(c) $0.9$

Show solution

The correct answer is (a) ★

The transition probability $P(Y{=}0|X{=}1)=0.2$ is given directly in the problem statement.

Exercise 1.24. A source emits symbols from $\{a,b,c,d\}$ uniformly at random. How many sequences of length $2$ have probability $\tfrac{1}{16}$?

(a) $4$
(b) $8$
(c) $16$

Show solution

The correct answer is (c) ★

Each symbol has probability $\tfrac{1}{4}$, so every length-$2$ sequence has probability $\tfrac{1}{4}\cdot\tfrac{1}{4}=\tfrac{1}{16}$. There are $4^2=16$ such sequences.

Exercise 1.25. For an i.i.d. ternary source with $P(0)=\tfrac{1}{2}$, $P(1)=P(2)=\tfrac{1}{4}$, which length-$3$ sequence is least probable: $000$, $012$, or $222$?

(a) $000$
(b) $012$
(c) $222$

Show solution

The correct answer is (c) ★

$P(000)=\tfrac{1}{8}$, $P(012)=\tfrac{1}{2}\cdot\tfrac{1}{4}\cdot\tfrac{1}{4}=\tfrac{1}{32}$, $P(222)=\tfrac{1}{64}$. The least probable is $222$.
