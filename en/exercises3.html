Exercises: 3. Prefix-free binary source codes

Exercise 3.1. Which of the following statements is correct? In your answer to this question, you must also explain why the others are incorrect.

(a) The code $\{a,b,c,ac\}$ is uniquely decodable.
(b) The code $\{aa,ab,ba,bb\}$ is not uniquely decodable.
(c) The code $\{aaa,aab,aac,bbb\}$ is prefix free.

Show solution

The correct answer is (c) ★

Code (a) is not uniquely decodable since the string $ac$ can be read as the single codeword $ac$ or as $a$ followed by $c$. Code (b) is prefix-free (equal-length distinct codewords) hence uniquely decodable, so the statement that it is not is false. Code (c) is prefix-free, the correct statement.

Exercise 3.2. Which of the following is a valid prefix-free code for the source?

(a) All binary sequences of $3$ bits.
(b) The code resulting from the concatenation of the code for $\{a,b,c\}$ given by $\{0,1,10\}$.
(c) A code given by different sequences of $4$ bits.

Show solution

The correct answer is (c) ★

The source takes values in the alphabet $$ \{aa,ab,ac,ba,bb,bc,ca,cb,cc\}, $$ hence a valid code consists of $9$ binary sequences. In (a), there are only $8$ different binary sequences of $3$ bits, which is insufficient. The code resulting from the concatenation of the code (b) is $$ \{00,01,010,10,11,110,100,101,1010\} $$ which is clearly not prefix free. Any code given by $9$ different sequences of $4$ bits is a prefix-free code.

Exercise 3.3. Which of the following statements is incorrect? In your answer to this question, you must also explain why the others are correct.

(a) The resulting code is prefix free with average length $\tfrac{10}{3}$ bits.
(b) The entropy of the source is $2\log_2(3)$ bits.
(c) The average length of the code is $1.05155\%$ higher than the entropy.

Show solution

The correct answer is (c) ★

The resulting code is given by $$ \{00,010,011,100,1010,1011,110,1110,1111\}, $$ which is prefix free. Since the source emits equiprobable symbols, each codeword is used with probability $\tfrac 19$ leading to an average length of $\tfrac{10}{3}$ bits. The entropy of the source is given by $$ \sum_{i=1}^9 p_i\log_2\tfrac{1}{p_i} = 9\cdot\tfrac 19\log_2(9)=\log_2(9)=2\log_2(3), $$ and the ratio between the average length and the entropy is $1.05155$, implying that the average length is $5.155\%$ higher than the entropy.

Exercise 3.4. What is now the average length of the code in question 5?

(a) The same as before.
(b) $\tfrac{59}{18}$ bits
(c) $\tfrac{8}{3}$ bits

Show solution

The correct answer is (b) ★

We have two ways of calculating the average length. Since the code is constructed as the concatenation $X=(X_1,X_2)$, the average length is given by $L_{\rm av}(X)=L_{\rm av}(X_1)+L_{\rm av}(X_2)$. The first symbol is encoded using the code $\{0,10,11\}$ with equal probability, hence $L_{\rm av}(X_1)=\tfrac 53$. The average length for the second emitted symbol depends on the first symbol: with probability $\tfrac 13$ (the first symbol is $a$) the average length of the second encoded symbol is $\tfrac 12\cdot 1 + \tfrac 14\cdot 2 + \tfrac 14\cdot 2 = \tfrac 32$; while with probability $\tfrac 23$ (the first symbol is either $b$ or $c$), the average length of the second encoded symbol is $\tfrac 53$. Altogether, we obtain $$ L_{\rm av}(X) = \frac 53 + \frac 13\cdot \frac 32 + \frac 23 \cdot \frac 53 = \frac{59}{18}. $$ The second way is to find the joint distribution of the source $U=(U_1,U_2)$ as $P_U(u)=P_{U_1}(u_1)P_{U_2|U_1}(u_2|u_1)$. For example, $P_U(aa)=P_{U_1}(a)P_{U_2|U_1}(a|a) = \tfrac 13\cdot \tfrac 12 = \tfrac 16$, or $P_U(ca)=P_{U_1}(c)P_{U_2|U_1}(a|c) = \tfrac 13\cdot \tfrac 13 = \tfrac 19$. Proceeding in the same way for all symbols in the source, we obtain the probability distribution $$ \biggl\{ \frac 16, \frac{1}{12},\frac{1}{12}, \frac 19, \frac 19,\frac 19,\frac 19,\frac 19,\frac 19 \biggr\} $$ Since the code has length distribution $$ \{2,3,3,3,4,4,3,4,4\}, $$ combining both results leads to the same answer.

Exercise 3.5.

A source $S$ has alphabet $\{a,b,c\}$ and probability distribution $\{0.4, 0.59999, 0.00001\}$. Consider the codes given by $\mathcal{C}_1=\{00, 11, 1010101010\}$ and $\mathcal{C}_2=\{000, 11, 101\}$. Indicate the true answer, justifying your choice and why the others are incorrect.

(a) $\mathcal{C}_1$ is more efficient than $\mathcal{C}_2$.
(b) One of the two codes is not prefix-free.
(c) For any sequence of three source symbols, the length of the encoded sequence using $\mathcal{C}_1$ will be shorter than the encoded sequence using $\mathcal{C}_2$.

Show solution

The correct answer is (a) ★

Answer (a) is true because $L_{\rm av}(\mathcal{C}_1) = 2.00008$ and $L_{\rm av}(\mathcal{C}_2) = 2.40001$. Answer (b) is false since both codes are prefix-free. Answer (c) is false, as for all sequences that contain symbol $c$ the code $\mathcal{C}_1$ will lead to a longer sequence than $\mathcal{C}_2$.

Exercise 3.6.

For $r\geq 2$, consider a source taking values in $\{1,\ldots,r\}$ and a code whose lengths are given by $\ell_j=\lceil \log_2(j+1)\rceil$, where $\lceil x \rceil$ is the ceiling operator. Indicate the false answer, justifying your choice and why the others are correct. You may use that $\lceil x \rceil < x + 1$.

(a) There exists a prefix-free code for any $r$.
(b) The value in Kraft's inequality satisfies $\sum_{j=1}^r 2^{-\ell_j} > \sum_{j=1}^r \frac{1}{2(j+1)}$.
(c) Some codewords can have the same length.

Show solution

The correct answer is (a) ★

Answer (a) is false because Kraft's inequality $\sum_{j=1}^r 2^{-\ell_j} \leq 1$ is satisfied only for $r=2$ and $r=3$. Answer (b) is true because, using that $\lceil \log_2(j+1)\rceil < \log_2(j+1)+1$, we have $$\sum_{j=1}^{r} 2^{-\lceil \log_2(j+1)\rceil} > \sum_{j=1}^{r}2^{-\log_2(j+1)-1} = \sum_{j=1}^{r}\frac{1}{2 (j+1)}.$$ Answer (c) is true because of the ceiling operator.

Exercise 3.7.

You are given a binary prefix-free code for a source $S$ that can take values $\{a,b,c,d,e,f\}$. It is known that $P_S(a)=P_S(b)=P_S(c)=\frac{1}{4}$ and that $P_S(d)=\frac{1}{8}$. Knowing that the code has a binary tree representation with no empty leaves, indicate which answer is false, justifying your choice and why the others are correct.

(a) For such code, for sure $L_{\rm av} > H(S)$.
(b) For such code, Kraft's inequality is satisfied with equality.
(c) The maximum length of a codeword for this code is at most 5.

Show solution

The correct answer is (a) ★

Answer (a) is false, since we can find a code such that $L_{\rm av}=H(S)$. Answer (b) is true since we are told that there are no empty leaves. Answer (c) is true, because knowing the alphabet size and the fact that there are no empty leaves, the tallest tree has depth five.

Exercise 3.8.

A source $S$ taking values in $\{1,2,3,4\}$ with probability distribution $\{\frac{1}{4}, \frac{1}{4}, \frac{3}{8}, \frac{1}{8}\}$ is compressed using a code with lengths $\{1,1,2,3\}$. Find the false answer, justifying your choice and why the others are correct.

(a) The code can not be uniquely decodable.
(b) The entropy of the source satisfies $H(S)<2$.
(c) It holds that $L_{\rm av}=\mathbb{E}[S]$, where $\mathbb{E}[S]$ is the expected value of the source.

Show solution

The correct answer is (c) ★

Answer (a) is correct since with two codewords of length 1 the code is necessarily not uniquely decodable. Answer (b) is correct as the entropy is $1.90564$. Finally, answer (c) is false because $\mathbb{E}[S]=\frac{19}{8}$ while $L_{\rm av}=\frac{13}{8}$.

Exercise 3.9.

A randomly generated binary source code with codewords $\{\boldsymbol{x}_1,\boldsymbol{x}_2\}$, such that each codeword is independently selected from the set $\{0,01,10\}$ with probability distribution $\bigl\{\tfrac12,\tfrac13,\tfrac16\bigr\}$, is used to represent a discrete memoryless source taking values in $\{a,b\}$ with probability distribution $\bigl\{\tfrac13,\tfrac23\bigr\}$. What is the probability of generating a uniquely decodable code with average length $\tfrac53$?

(a) $0$
(b) $\tfrac14$
(c) $\tfrac19$
(d) $\tfrac{1}{12}$

Show solution

The correct answer is (b) ★

From all possible randomly generated codes, only $\{0,01\}$ and $\{0,10\}$ satisfy both conditions on unique decodability and average length. The probabilities of such codes are $\tfrac 16$ and $\tfrac{1}{12}$, respectively, whose sum leads to the correct answer.

Exercise 3.10.

A discrete source takes values in the set $\{a,b,c\}$ with probabilities $\{p,2p,3p\}$ for some parameter $p\neq 0$. What is the average length of the optimal binary prefix-free code for this source?

(a) $\log_2(3)$
(b) $\tfrac 32$
(c) $2$
(d) $\tfrac{4+3\log_2(3)}{6}$

Show solution

The correct answer is (b) ★

Only $p=\tfrac 16$ leads to a valid probability distribution. The Huffman code for the source has lengths $\{2,2,1\}$, so $L_{\rm av}=9p=\tfrac 32$.

Exercise 3.11.

For a given discrete source $X$, the optimal binary prefix-free code is given by $\{0,10,11\}$ and satisfies $L_{\rm av}=H(X)$. A new source is constructed as $Y=(X_1,X_2)$, where $X_1$ and $X_2$ are independent and both have the same distribution as $X$. What is the average length of the optimal binary prefix-free code for $Y$?

(a) $\log_2(9)$
(b) $\frac{10}{3}$
(c) $3$
(d) $4$

Show solution

The correct answer is (c) ★

If $L_{\rm av}=H(X)$, then the distribution of $X$ is given by powers of $2$, that is, $\bigl\{ \frac 12, \frac 14, \frac 14 \bigr\}$. Since the source is memoryless, we can calculate the probabilities of $Y$, find the optimal (Huffman) code of $Y$, and obtain its average length.

Exercise 3.12.

A binary code $\{0,10,110,111\}$ is used for a source with $4$ symbols. Is this code prefix-free?

(a) Yes
(b) No, because $0$ is a prefix of $10$
(c) No, because $110$ is a prefix of $111$

Show solution

The correct answer is (a) ★

No codeword is a prefix of another: $0$ is not a prefix of $10$, since the first bits differ. Checking all pairs confirms prefix-freeness.

Exercise 3.13. What is the Kraft value of the code $\{0,10,110,111\}$?

(a) $1$
(b) $\tfrac{7}{8}$
(c) $\tfrac{3}{4}$

Show solution

The correct answer is (a) ★

The Kraft value is $2^{-1}+2^{-2}+2^{-3}+2^{-3}=\tfrac{1}{2}+\tfrac{1}{4}+\tfrac{1}{8}+\tfrac{1}{8}=1$.

Exercise 3.14. A source has entropy $H(X)=2.5$ bits. What can be said about the average length $L_{\rm av}$ of any binary prefix-free code for this source?

(a) $L_{\rm av}\geq 2.5$
(b) $L_{\rm av}\leq 2.5$
(c) $L_{\rm av}= 2.5$

Show solution

The correct answer is (a) ★

By the source coding theorem, the average length of any uniquely decodable (hence prefix-free) code satisfies $L_{\rm av}\geq H(X)=2.5$ bits.

Exercise 3.15. Is the code $\{0,01,011\}$ uniquely decodable?

(a) No
(b) Yes, it is prefix-free
(c) Yes, but not prefix-free

Show solution

The correct answer is (a) ★

The codeword $0$ is a prefix of $01$, and $01$ is a prefix of $011$. The sequence $011$ can be decoded as either the third symbol or the first followed by the second, so it is not uniquely decodable.

Exercise 3.16. Does there exist a binary prefix-free code with codeword lengths $\{1,2,2,3\}$?

(a) Yes, because the Kraft inequality is satisfied
(b) No, the Kraft inequality is violated
(c) Only if the source has $4$ symbols

Show solution

The correct answer is (b) ★

The Kraft value is $2^{-1}+2^{-2}+2^{-2}+2^{-3}=\tfrac{4+2+2+1}{8}=\tfrac{9}{8}>1$, so the Kraft inequality is violated and no such code exists.

Exercise 3.17. A binary source code $\{00,01,10,11\}$ is used for a source with $4$ equiprobable symbols. What is the average length?

(a) $2$ bits
(b) $1$ bit
(c) $4$ bits

Show solution

The correct answer is (a) ★

Each codeword has length $2$ and all symbols are equiprobable, so $L_{\rm av}=2$ bits.

Exercise 3.18.

For a source with $r$ equiprobable symbols, the entropy is $\log_2(r)$ bits. What is the minimum possible average length of a binary fixed-length prefix-free code?

(a) $\lceil\log_2(r)\rceil$ bits
(b) $\log_2(r)$ bits exactly
(c) $r$ bits

Show solution

The correct answer is (a) ★

A fixed-length binary code must assign the same number of bits to each of the $r$ symbols, and $\lceil\log_2(r)\rceil$ bits suffice to index them all, so the minimum fixed length is $\lceil\log_2(r)\rceil$ bits. When $r$ is a power of $2$ this equals $\log_2(r)$; otherwise $\lceil\log_2(r)\rceil>\log_2(r)$. Using $\log_2(r)$ exactly is in general impossible with a fixed integer length, and $r$ bits is far larger than necessary.

Exercise 3.19. A code for a source $\{a,b,c\}$ has codewords $\{0,10,11\}$. What is the Kraft value?

(a) $1$
(b) $\tfrac{3}{4}$
(c) $\tfrac{1}{2}$

Show solution

The correct answer is (a) ★

The Kraft value is $2^{-1}+2^{-2}+2^{-2}=\tfrac{1}{2}+\tfrac{1}{4}+\tfrac{1}{4}=1$.

Exercise 3.20. Which of the following codes is uniquely decodable but NOT prefix-free?

(a) $\{0,01,11\}$
(b) $\{0,10,11\}$
(c) $\{0,01,10\}$

The code $\{0,01,11\}$ is not prefix-free since $0$ is a prefix of $01$. However, it can be shown to be uniquely decodable. The code $\{0,10,11\}$ is prefix-free.

Exercise 3.21.

A source takes values in $\{a,b\}$ with $P(a)=p$ and $P(b)=1-p$. For what value of $p$ does the optimal binary prefix-free code achieve $L_{\rm av}=H(X)$?

(a) $p=\tfrac{1}{2}$
(b) Any $p$
(c) $p=\tfrac{1}{4}$

Show solution

The correct answer is (a) ★

For a binary source, any prefix-free code assigns length $1$ to both symbols, so $L_{\rm av}=1$ always. The entropy $H(X)=h_2(p)=1$ only when $p=\tfrac{1}{2}$.

Exercise 3.22.

A binary code with codeword lengths $\{\ell_1,\ell_2\}$ encodes a two-symbol source with the uniform distribution $\{\tfrac12,\tfrac12\}$, achieving an average length of $\tfrac52$. The first codeword has length $\ell_1=1$.

What is the length $\ell_2$ of the second codeword?

(a) $\ell_2=4$
(b) $\ell_2=5$
(c) $\ell_2=2$
(d) $\ell_2=3$

Show solution

The correct answer is (a) ★

The average length is $L_{\rm av}=\tfrac12\ell_1+\tfrac12\ell_2$. With $\ell_1=1$ and $L_{\rm av}=\tfrac52$, we get $\tfrac12(1)+\tfrac12\ell_2=\tfrac52$, so $\tfrac12\ell_2=2$ and $\ell_2=4$. (Note $5$ comes from forgetting to subtract $\ell_1$.)

Exercise 3.23.

A binary code has codeword lengths $\{1,4\}$ (one codeword of length $1$, one of length $4$). It is used over a two-symbol source with distribution $\{\tfrac13,\tfrac23\}$, assigning the shorter codeword to the more probable symbol to minimize the average length.

What is the resulting average length?

(a) $2$ bits
(b) $3$ bits
(c) $\tfrac52$ bits
(d) $\tfrac53$ bits

Show solution

The correct answer is (a) ★

Assign the length-$1$ codeword to the probability-$\tfrac23$ symbol and the length-$4$ codeword to the probability-$\tfrac13$ symbol: $L_{\rm av}=\tfrac23\cdot1+\tfrac13\cdot4=\tfrac23+\tfrac43=\tfrac63=2$. The swapped (suboptimal) assignment $\tfrac13\cdot1+\tfrac23\cdot4=3$ is larger, so $2$ bits is the minimal value.

Exercise 3.24.

Consider three statements about binary source codes. Exactly one is false. Which statement is the false one?

  1. The code $\{0,\,01,\,001\}$ is not prefix-free but is uniquely decodable.
  2. The code $\{1,\,01,\,001\}$ is prefix-free and uniquely decodable.
  3. The code $\{0,\,1,\,01\}$ is not uniquely decodable.

(a) Statement 1: $\{0,\,01,\,001\}$ is not prefix-free but uniquely decodable
(b) Statement 2: $\{1,\,01,\,001\}$ is prefix-free and uniquely decodable
(c) Statement 3: $\{0,\,1,\,01\}$ is not uniquely decodable
(d) None of them is false; all three are true

Show solution

The correct answer is (a) ★

In $\{0,\,01,\,001\}$, the word $0$ is a prefix of the others, so it is correctly not prefix-free. However, concatenating $0$ and $01$ gives $0\cdot 01 = 001$, which equals the third codeword, so $001$ has two decodings (symbols 1,2 or symbol 3). Hence it is not uniquely decodable, making statement 1 false. (Statement 3 is true by the same kind of collision $0\cdot 1 = 01$; statement 2 is true since each word is leading $0$s ended by a single $1$, hence prefix-free $\Rightarrow$ uniquely decodable.)

Exercise 3.25.

A binary prefix-free code is represented by a binary tree of depth $5$. The tree has exactly one empty leaf (a leaf position not assigned to any codeword), located at depth $5$. Compute the Kraft value $\displaystyle\sum_{i=1}^{r} 2^{-\ell_i}$, where $\ell_i$ are the depths of the occupied leaves.

(a) $\dfrac{31}{32} = 1 - 2^{-5}$
(b) $1$
(c) $\dfrac{15}{16} = 1 - 2^{-4}$
(d) $\dfrac{1}{32} = 2^{-5}$

Show solution

The correct answer is (a) ★

A full binary tree, where every leaf carries the weight $2^{-(\text{depth})}$, sums to exactly $1$ (each parent's weight equals its two children's). Here one leaf at depth $5$ is empty, so it contributes $2^{-5}$ to the full-tree total of $1$ but is not counted as a codeword. Thus $\sum_i 2^{-\ell_i} = 1 - 2^{-5} = 1 - \tfrac{1}{32} = \tfrac{31}{32}$.

Exercise 3.26.

A binary prefix-free code with $3$ codewords satisfies Kraft's inequality with value $\frac{7}{8}$, that is $\sum_{i=1}^{3} 2^{-\ell_i}=\frac{7}{8}$. The code encodes a source whose $3$ symbols are equiprobable. Calculate the average codeword length $L_{\mathrm{av}}$.

(a) $L_{\mathrm{av}}=2$ bits per symbol
(b) $L_{\mathrm{av}}=\log_2 3\approx 1.585$ bits per symbol
(c) $L_{\mathrm{av}}=\frac{7}{8}$ bits per symbol
(d) $L_{\mathrm{av}}=3$ bits per symbol

Show solution

The correct answer is (a) ★

The lengths are forced by $2^{-\ell_1}+2^{-\ell_2}+2^{-\ell_3}=\frac{7}{8}$: exactly one codeword has length $1$ (two would already give $1>\frac78$, none would give at most $\frac34<\frac78$), and the remaining $\frac78-\frac12=\frac38=\frac14+\frac18$ forces lengths $2$ and $3$. Hence the multiset is $\{1,2,3\}$. With a uniform source $\mathbb{P}(x_i)=\frac13$, so $L_{\mathrm{av}}=\frac13(1+2+3)=\frac63=2$ bits per symbol.

Exercise 3.27.

A binary prefix-free code is used over a source with a uniform distribution and achieves an average codeword length of $L_{\rm av}=\frac{39}{11}$ bits. What is the maximum possible size $M$ of the source alphabet?

(a) $M=11$
(b) $M=12$
(c) $M=10$
(d) $M=8$

Show solution

The correct answer is (a) ★

For a uniform source $H(U)=\log_2 M$, and any prefix-free code satisfies $L_{\rm av}\ge H(U)$. Thus $\frac{39}{11}\ge\log_2 M$, giving $M\le 2^{39/11}=2^{3.545\ldots}\approx 11.68$, so $M\le 11$. The value $M=11$ is attainable: 5 codewords of length $3$ and 6 of length $4$ give $\sum_i 2^{-\ell_i}=\tfrac58+\tfrac{6}{16}=1$ (Kraft holds) and $L_{\rm av}=\frac{5\cdot3+6\cdot4}{11}=\frac{39}{11}$. Since $\log_2 12=3.585>3.545$, $M=12$ is impossible. Hence $M_{\max}=11$.

Exercise 3.28.

A source $U$ is uniform on $\{1,2,3\}$ and is encoded as $1\mapsto\texttt{1}$, $2\mapsto\texttt{10}$, $3\mapsto\texttt{11}$. Let $\ell(U)$ be the codeword length in bits and let $H(U)$ be the entropy in bits. Compute $\mathbb{P}\big[\ell(U)\ge H(U)\big]$.

(a) $\tfrac23$
(b) $\tfrac13$
(c) $1$
(d) $\log_2 3$

Show solution

The correct answer is (a) ★

The lengths are $\ell(1)=1$ and $\ell(2)=\ell(3)=2$, so $\ell(U)=1$ with probability $\tfrac13$ and $\ell(U)=2$ with probability $\tfrac23$. The entropy is $H(U)=\log_2 3\approx 1.585$ bits. Since $1<1.585$ but $2\ge 1.585$, the event $\{\ell(U)\ge H(U)\}$ equals $\{\ell(U)=2\}=\{U\in\{2,3\}\}$. Hence the probability is $\tfrac23$.

Exercise 3.29.

Consider a binary code with length distribution $\{1,2,3,3,5,5\}$. Is the claim "there exists a prefix-free binary code with these lengths" true or false, and why?

(a) False: $K=\frac{17}{16}>1$, so Kraft fails and no such code exists
(b) True: $K=1$ exactly, so a prefix-free code exists
(c) True: $K=\frac{15}{16}<1$, so a prefix-free code exists
(d) False: $K=\frac{17}{16}>1$, and no code of any kind with these lengths exists

Show solution

The correct answer is (a) ★

Compute the Kraft sum: $K=2^{-1}+2^{-2}+2^{-3}+2^{-3}+2^{-5}+2^{-5}=\frac{16+8+4+4+1+1}{32}=\frac{34}{32}=\frac{17}{16}=1.0625$. Since $K>1$, the disjoint leaf regions would exceed the whole tree, so no prefix-free code with these lengths exists. The claim is false. (Option 3 is wrong because non-prefix-free codes with arbitrary lengths always exist.)

Exercise 3.30.

A given binary code has length distribution $\{1,2,2\}$. Is the claim "this code is surely prefix-free" true or false?

(a) False: $K=1$ only guarantees some prefix-free code exists; e.g. $\{0,00,01\}$ has these lengths but is not prefix-free
(b) True: since $K=1\le 1$, any code with these lengths is prefix-free
(c) False: $K=\frac{3}{2}>1$, so no code with these lengths can be prefix-free
(d) True: $\{0,10,11\}$ is prefix-free, so every code with these lengths is too

Show solution

The correct answer is (a) ★

Kraft gives $K=2^{-1}+2^{-2}+2^{-2}=\frac{1}{2}+\frac{1}{4}+\frac{1}{4}=1\le 1$, so some prefix-free code with these lengths exists (e.g. $\{0,10,11\}$). But Kraft is a condition on the multiset of lengths, not on a particular code. The counterexample $\{0,00,01\}$ has lengths $\{1,2,2\}$ yet $0$ is a prefix of $00$ and $01$, so it is not prefix-free. The claim is false.

Exercise 3.31.

Consider the length distribution $\{1,2,3,3,3\}$. Is the claim "we cannot find a prefix-free binary code with these lengths" true or false?

(a) True: $K=\frac{9}{8}>1$, so Kraft fails and no prefix-free code exists
(b) False: $K=1$, so a prefix-free code does exist
(c) False: $K=\frac{7}{8}<1$, so a prefix-free code does exist
(d) True: $K=\frac{9}{8}>1$, but only because the three length-3 words clash

Show solution

The correct answer is (a) ★

Kraft sum: $K=2^{-1}+2^{-2}+2^{-3}+2^{-3}+2^{-3}=\frac{4+2+1+1+1}{8}=\frac{9}{8}=1.125$. Since $K>1$, the disjoint leaf regions would cover more than the whole tree, which is impossible, so no prefix-free code with these lengths exists. The claim that none can be found is therefore true.

Exercise 3.32.

A binary prefix-free code has $4$ codewords with positive-integer lengths $\ell_1,\ell_2,\ell_3,\ell_4$ satisfying Kraft's equality $\sum_{i=1}^{4} 2^{-\ell_i} = \frac{15}{16}$. What is the (unique) set of four lengths?

(a) $\{1,2,3,4\}$
(b) $\{2,2,3,4\}$
(c) $\{1,2,2,4\}$
(d) $\{2,3,4,4\}$

Show solution

The correct answer is (a) ★

Multiply by $16=2^4$: the condition becomes $\sum_i 2^{4-\ell_i}=15$. Any $\ell_i\ge 5$ adds a non-integer fraction, so all $\ell_i\le 4$, and the terms $a_i=2^{4-\ell_i}$ are coins from $\{8,4,2,1\}$ that must sum to $15$. The only length-$4$ multiset of these coins summing to $15$ is one of each, $\{8,4,2,1\}$, giving lengths $\{1,2,3,4\}$. Check: $2^{-1}+2^{-2}+2^{-3}+2^{-4}=\frac{15}{16}$.

Exercise 3.33.

For the code with $\sum_{i=1}^{4} 2^{-\ell_i} = \frac{15}{16}$ (positive-integer lengths), how many codewords have length exactly $4$?

(a) Exactly one
(b) None
(c) Exactly two
(d) It cannot be determined

Show solution

The correct answer is (a) ★

Writing $a_i=2^{4-\ell_i}\in\{8,4,2,1\}$ with $\sum a_i=15$, a length-$4$ codeword corresponds to a coin of value $1$ (the only odd coin). If zero coins equalled $1$, the remaining even coins $\{2,4,8\}$ would sum to an even number, never the odd total $15$; and the multiset is forced uniquely to $\{8,4,2,1\}$, ruling out two or more. Hence exactly one codeword has length $4$.

Exercise 3.34.

A binary prefix-free source code has the nine codeword lengths $\{1,3,3,4,4,5,5,5,5\}$. It is known that the code's average length equals the entropy of the source. Calculate this average length (in bits per symbol).

(a) $\dfrac{19}{8}=2.375$ bits/symbol
(b) $\dfrac{34}{9}\approx 3.778$ bits/symbol
(c) $\dfrac{27}{8}=3.375$ bits/symbol
(d) $3.5$ bits/symbol

Show solution

The correct answer is (a) ★

Equality $L_{\rm av}=H(U)$ holds iff the source is dyadic with $p_i=2^{-\ell_i}$ (the relative entropy $\sum_i p_i\log_2\frac{p_i}{2^{-\ell_i}}$ vanishes). The Kraft sum $\tfrac12+2\cdot\tfrac18+2\cdot\tfrac1{16}+4\cdot\tfrac1{32}=1$ confirms validity. Then $L_{\rm av}=\sum_i 2^{-\ell_i}\ell_i=\tfrac12\cdot1+2\cdot\tfrac18\cdot3+2\cdot\tfrac1{16}\cdot4+4\cdot\tfrac1{32}\cdot5=\tfrac{19}{8}=2.375$ bits/symbol. (The value $\tfrac{34}{9}$ is an arbitrary distractor.)
