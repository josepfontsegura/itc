Resit exam 2025

Problem 1. An experiment consists of the following two steps. First, flip a fair coin and record heads or tails. If the result is heads, roll a fair six-sided die and record its face; if the result is tails, flip a second fair coin and record its outcome. What is the entropy in bits of the outcome of the two-step experiment?

(a) $\tfrac{1+\log_2(6)}{2}$
(b) $\tfrac{3+\log_2(3)}{2}$
(c) $\tfrac{4+\log_2(3)}{2}$
(d) $\tfrac{2+\log_2(6)}{2}$

Show solution

The correct answer is (c) ★

The result can be modelled by $X=(A,B)$. After determining all outcomes, $X$ has distribution $\bigl\{\tfrac{1}{12}, \tfrac{1}{12}, \tfrac{1}{12}, \tfrac{1}{12}, \tfrac{1}{12}, \tfrac{1}{12}, \tfrac{1}{4}, \tfrac{1}{4}\bigr\}$. Using the chain rule $H(X)=H(A)+H(B|A)$, with $H(A)=1$ and $H(B|A)=\tfrac{1+\log_2(6)}{2}$.

Problem 2. A discrete source takes values in the set $\{a,b,c\}$ with probabilities $\{p,2p,3p\}$ for some parameter $p\neq 0$. What is the average length of the optimal binary prefix-free code for this source?

(a) $\log_2(3)$
(b) $\tfrac 32$
(c) $2$
(d) $\tfrac{4+3\log_2(3)}{6}$

Show solution

The correct answer is (b) ★

Only $p=\tfrac 16$ leads to a valid probability distribution. The Huffman code for the source has lengths $\{2,2,1\}$, so $L_{\rm av}=9p=\tfrac 32$.

Problem 3. A discrete memoryless source $W_n=(X_1,\ldots,X_n)$, where each $X_i\in\{a,b,c,d,e,f\}$ with equal probability, is encoded using the optimal binary prefix-free code. In which of the following intervals does the average length per symbol of the optimal binary prefix-free code for $W_{3}$ lie?

(a) $\bigl[\tfrac{3+3\log_2(3)}{2},\tfrac{3\log_2(3) + 4}{2} \bigr)$
(b) $\bigl[\tfrac{1+\log_2(3)}{2},\tfrac{3\log_2(3) + 4}{6} \bigr)$
(c) $\bigl[1+\log_2(3),\tfrac{3\log_2(3) + 4}{3} \bigr)$
(d) $\bigl[3+3\log_2(3),3\log_2(3) + 4 \bigr)$

Show solution

The correct answer is (c) ★

The average length per symbol lies in $\bigl[H(X_1),H(X_1)+\tfrac 13\bigr)$. Using $H(X_1)=\log_2(6)$, we obtain the result.

Problem 4. Define the random variable $Z$ as $Z=\log_2 P_X(X)$, where $X$ is a discrete memoryless source $X=(U_1,\ldots,U_r)$, with each $U_i$ taking values in the set $\{1,2,3,\ldots,n\}$ with equal probability. What is the expected value of $Z$?

(a) $r\log_2(n)$
(b) $-r\log_2(n)$
(c) $n\log_2(r)$
(d) $-n\log_2(r)$

Show solution

The correct answer is (b) ★

Since $P_X(x)=\tfrac{1}{n^r}$ for all $x$, the random variable is deterministic with value $\log_2(n^{-r})$.

Problem 5. The codewords of a given $(5,1)$ linear code with weight distribution satisfying $A_2=1$ are transmitted with equal probability over a binary symmetric channel with bit flip probability $p$. What is the probability of receiving the all-ones sequence?

(a) $\tfrac{p^5-2p^4 +2p^3}{2}$
(b) $\tfrac{2p^5 -2p^4 +p^3}{2}$
(c) $\tfrac{p^5-p^4+p^3}{2}$
(d) $\tfrac{p^5+p^4+p^3}{2}$

Show solution

The correct answer is (b) ★

The code has two codewords, $00000$ and a second codeword with $2$ ones and $3$ zeros. Then, the probability of receiving $11111$ is $\tfrac 12 p^5 + \tfrac 12 p^3(1-p)^2$. Expanding and simplifying gives the result.

Problem 6. The code $\{000,111\}$ is transmitted over a binary channel with conditional probabilities $P_{Y|X}(0|0)=1$ and $P_{Y|X}(0|1)=\tfrac 12$. The receiver uses a minimum likelihood decoding rule, that is $\hat m = \arg\min_m P_{\boldsymbol{Y}|\boldsymbol{X}}(\boldsymbol{y}|\boldsymbol{x}_m)$, to decode the channel output. What is the error region of the second message?

(a) $\{001,010,011,100,101,110,111\}$
(b) $\{000\}$
(c) $\{000,001,010,100\}$
(d) $\{011,101,110,111\}$

Show solution

The correct answer is (a) ★

Since $P_{Y|X}(1|0)=0$, we have $P_{\boldsymbol{Y}|\boldsymbol{X}}(\boldsymbol{y}|000)=0$ for any output with at least one $1$. Hence, $\hat m = 1$ for all outputs except $\boldsymbol{y}=000$, for which $\hat m = 2$ — this is the only output for which the second message is decoded correctly.

Problem 7. Two random variables $X$ and $Y$ take values in $\{a,b\}$ and $\{a,b,c,d\}$, respectively, such that $X$ has equiprobable values and that $H(Y|X)=\tfrac 32$. Which of the following relations is false?

(a) $H(X,Y) = 3$
(b) $H(Y) + H(X|Y) = \tfrac{5}{2}$
(c) $I(X;Y) = 1 - H(X|Y)$
(d) $H(Y) \geq \tfrac 32$

Show solution

The correct answer is (a) ★

The statement $I(X;Y) = 1 - H(X|Y)$ is true because $H(X)=1$. $H(Y)\geq \tfrac 32$ is true because conditioning reduces entropy. $H(Y) + H(X|Y) = H(X,Y) = H(X)+H(Y|X) = \tfrac 52$. The joint entropy is not $3$.
