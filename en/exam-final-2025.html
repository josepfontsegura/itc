Final exam 2025

Problem 1. A random word generator emits words of $r$ letters, where each letter belongs to an alphabet of size $n$. What is the maximum binary entropy of the random word generator?

(a) $\log_2(r)$
(b) $\log_2(n)$
(c) $r\log_2(n)$
(d) $n\log_2(r)$

Show solution

The correct answer is (c) ★

The random word generator can be described as a discrete source $X=(U_1,\ldots,U_r)$ where each $U_i$ takes values in an alphabet of cardinality $n$. The entropy of $X$ satisfies $H(X)\leq\log_2|\mathcal{X}|$, where $|\mathcal{X}|=n^r$.

Problem 2. A randomly generated binary source code with codewords $\{\boldsymbol{x}_1,\boldsymbol{x}_2\}$, such that each codeword is independently selected from the set $\{0,01,10\}$ with probability distribution $\bigl\{\tfrac12,\tfrac13,\tfrac16\bigr\}$, is used to represent a discrete memoryless source taking values in $\{a,b\}$ with probability distribution $\bigl\{\tfrac13,\tfrac23\bigr\}$. What is the probability of generating a uniquely decodable code with average length $\tfrac53$?

(a) $0$
(b) $\tfrac14$
(c) $\tfrac19$
(d) $\tfrac{1}{12}$

Show solution

The correct answer is (b) ★

From all possible randomly generated codes, only $\{0,01\}$ and $\{0,10\}$ satisfy both conditions on uniquely decodability and average length. The probability of such codes is $\tfrac 16$ and $\tfrac{1}{12}$, respectively, whose sum leads to the correct answer.

Problem 3. Consider a source $W_n$ that emits sequences of $n$ i.i.d. bits such that the probability of $1$ is twice the probability of $0$. The average length of the optimal binary prefix-free code to represent $W_n$ is denoted as $L_{\rm av}(W_n)$. What is the value of the following ratio?$$\frac{L_{\rm av}(W_1)}{L_{\rm av}(W_2)}$$

(a) $1$
(b) $\tfrac{9}{23}$
(c) $\tfrac{9}{17}$
(d) $\tfrac12$

Show solution

The correct answer is (c) ★

Clearly $L_{\rm av}(W_1)=1$. Using that each bit takes values $\{0,1\}$ with probability distribution $\bigl\{\tfrac13,\tfrac23\bigr\}$, we obtain that $W_2\in\{00,01,10,11\}$ with probabilities $\bigl\{\tfrac19,\tfrac29,\tfrac29,\tfrac49\bigr\}$. The Huffman code for $W_2$ has length distribution $\{3,3,2,1\}$ leading to $L_{\rm av}(W_2)=\tfrac{17}{9}$.

Problem 4. A discrete memoryless source $W_n=(X_1,\ldots,X_n)$, where each $X_i$ takes values in $\{a,b,c\}$ with probability distribution $\bigl\{\tfrac12,\tfrac13,\tfrac16\}$, is represented using two coding schemes. In coding scheme (a), we design the optimal binary prefix-free code for $X_1$ and then represent each symbol $X_i$ with the same code. In coding scheme (b), we design the optimal binary prefix-free code for $W_n$. The average length to represent $W_n$ using the coding schemes (a) and (b) is respectively denoted as $L_{\rm av}^\text{(a)}(W_n)$ and $L_{\rm av}^\text{(b)}(W_n)$. What is the value of the following limit?$$\lim_{n\to\infty} \frac{L_{\rm av}^\text{(b)}(W_n)}{L_{\rm av}^\text{(a)}(W_n)}$$

(a) $\tfrac{4+3\log_2(3)}{10}$
(b) $\tfrac{4+3\log_2(3)}{9}$
(c) $\tfrac{4+3\log_2(3)}{6}$
(d) $\tfrac{4+3\log_2(3)}{11}$

Show solution

The correct answer is (b) ★

The Huffman code for $X_1$ has length distribution $\{1,2,2\}$ and average length $L_{\rm av}(X_1)=\tfrac32$. In coding scheme (a), we have $L_{\rm av}(W_n) = \sum_{i=1}^n L_{\rm av}(X_i) = \tfrac{3n}{2}$ due to concatenating the same code. On the other hand, as $n\to\infty$ the average length of the Huffman code for $W_n$ satisfies $\frac1n L_{\rm av}(W_n) = H(X_1)$. Combining both facts and using that $H(X_1)= \tfrac 23 + \tfrac{\log_2(3)}{2}$, we obtain the correct answer.

Problem 5. The codewords of a binary linear code with generating matrix$$\begin{pmatrix}0 & 1\\1 & 0\end{pmatrix}$$are transmitted with equal probability over a binary channel with conditional probabilities $P_{Y|X}(0|0)=0$ and $P_{Y|X}(0|1)=\tfrac13$. The receiver uses a maximum Hamming distance rule to decode the channel output. What is the error probability of the code?

(a) $\tfrac{11}{36}$
(b) $\tfrac{5}{9}$
(c) $\tfrac{8}{9}$
(d) $\tfrac{35}{36}$

Show solution

The correct answer is (b) ★

Since the generated code is $\{00,10,01,11\}$ and the receiver uses a maximum Hamming distance rule, the decoder $\hat m(\boldsymbol{y})$ is such that $\hat m(00)=4$, $\hat m(01)=2$, $\hat m(10)=3$ and $\hat m(11)=1$. Hence, $P_{\rm e}(1)=0$, $P_{\rm e}(2)=\tfrac 23$, $P_{\rm e}(3)=\tfrac 23$ and $P_{\rm e}(4)=\tfrac89$. Using that the four codewords are equiprobable, we obtain the result.

Problem 6. A binary channel has input $X$ taking values in $\{0,1\}$ with equal probability, output $Y$ taking values in $\{0,\star,1\}$, and conditional probabilities $P_{Y|X}(y|x)=\tfrac13$ for all $x$ and $y$. What is the value of the conditional entropy $H(X|Y)$?

(a) $1$
(b) $\tfrac{3\log_2(3)-1}{3}$
(c) $\tfrac{3\log_2(3)-2}{3}$
(d) $0$

Show solution

The correct answer is (a) ★

Since $H(Y|X)=H(Y)=\log_2(3)$, the mutual information of the channel satisfies $I(X;Y)=0$, which also implies that $H(X|Y)=H(X)=1$.

Problem 7. Three random variables $X_1$, $X_2$ and $X_3$, form a Markov chain $X_1\rightarrow X_2\rightarrow X_3$, where in addition $X_2$ is a deterministic function of $X_1$. What is the joint entropy of the three random variables?

(a) $H(X_1)+H(X_2)+H(X_3)$
(b) $H(X_1)+H(X_3)$
(c) $H(X_2)+H(X_3|X_2)$
(d) $H(X_1)+H(X_3|X_2)$

Show solution

The correct answer is (d) ★

The joint entropy can be expanded as $H(X_1)+H(X_2|X_1)+H(X_3|X_1,X_2)$. Since $X_2$ is a deterministic function of $X_1$, we have that $H(X_2|X_1)=0$. Also, conditioned on $X_2$, $X_1$ and $X_3$ become independent so that $H(X_3|X_1,X_2)=H(X_3|X_2)$. Combining both observations, we obtain the correct answer.
