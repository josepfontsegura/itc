Exercise 2.1.

Let $(X,Y)$ have joint distribution $P_{XY}(1,0)=\tfrac{3}{10}$, $P_{XY}(0,1)=\tfrac{1}{5}$, $P_{XY}(0,0)=\tfrac{1}{10}$ (so $P_{XY}(1,1)=\tfrac{2}{5}$ by normalization). Which of the following is correct for $\mathbb{E}[X]$ and $\mathbb{E}[Y]$?

(a) $\mathbb{E}[X] = \tfrac{7}{10}$ and $\mathbb{E}[Y] = \tfrac{2}{5}$
(b) $\mathbb{E}[X] = \tfrac{2}{5}$ and $\mathbb{E}[Y] = \tfrac{7}{10}$
(c) $\mathbb{E}[X] = \tfrac{7}{10}$ and $\mathbb{E}[Y] = \tfrac{3}{5}$

Show solution

The correct answer is (c) ★

By normalization, we deduce that $P_{XY}(1,1) = \tfrac{2}{5}$. Since $X$ and $Y$ are binary, $\mathbb{E}[X] = P_X(1)$ and $\mathbb{E}[Y] = P_Y(1)$. Using $P_X(1) = P_{XY}(1,0) + P_{XY}(1,1) = \tfrac{3}{10} + \tfrac{2}{5} = \tfrac{7}{10}$ and $P_Y(1) = P_{XY}(0,1) + P_{XY}(1,1) = \tfrac{1}{5} + \tfrac{2}{5} = \tfrac{3}{5}$, we find the correct answer.

Exercise 2.2.

Let $(X,Y)$ have $P_{XY}(0,0)=\tfrac{1}{10}$, $P_{XY}(0,1)=\tfrac{1}{5}$, $P_{XY}(1,0)=\tfrac{3}{10}$, $P_{XY}(1,1)=\tfrac{2}{5}$, and let $Z=\mathbb{1}\{X\neq Y\}$. What are the alphabet and probability distribution of $Z$, respectively?

(a) $\{0,1\}$ and $\bigl\{\tfrac12, \tfrac12\bigr\}$
(b) $\{0,1,2\}$ and $\bigl\{\tfrac{1}{10}, \tfrac{1}{2}, \tfrac{2}{5}\bigr\}$
(c) $\{0,1\}$ and $\bigl\{\tfrac{2}{5}, \tfrac{3}{5}\bigr\}$

Show solution

The correct answer is (a) ★

Since $Z = 0$ when $X=Y$, that is $(0,0)$ or $(1,1)$, we get $P_Z(0) = P_{XY}(0,0) + P_{XY}(1,1) = \tfrac{1}{10} + \tfrac{2}{5} = \tfrac{1}{10} + \tfrac{4}{10} = \tfrac{5}{10} = \tfrac12$. Also, $Z=1$ when $X\neq Y$, that is $(0,1)$ or $(1,0)$, hence $P_Z(1) = \tfrac{1}{5} + \tfrac{3}{10} = \tfrac{2}{10} + \tfrac{3}{10} = \tfrac{5}{10} = \tfrac12$. Therefore, $Z\in\{0,1\}$ with equal probability.

Exercise 2.3.

Let $Z\in\{0,1\}$ with $P_Z(0)=P_Z(1)=\tfrac12$. What is the value of $\mathbb{E}[\log_2(1+Z)]$?

(a) $\tfrac{1}{2}$
(b) $\tfrac{13}{10}$
(c) $1+\log_2(3)$

Show solution

The correct answer is (a) ★

Since $Z$ is $0$ or $1$ equiprobably, $\mathbb{E}[\log_2(1+Z)] = \log_2(1)\cdot\tfrac12 + \log_2(2)\cdot\tfrac12 = \tfrac12$.

Exercise 2.5.

Let $U$ be uniform on $\mathcal{U}$ with $|\mathcal{U}|=n$ (so $P_U(u)=1/n$ for all $u$), and let $g(x)=-\tfrac12\log_2(x)$ (so $g(2^n)=-n/2$). Which of the following statements is true? You can reason by elimination.

(a) $|\mathcal{U}| = 2^n$
(b) $\displaystyle\lim_{n\to\infty} \frac{g(2^n)}{P_U(2^n)} = -\infty$
(c) $\displaystyle\lim_{n\to\infty} n P_U(u) = 0$

Show solution

The correct answer is (b) ★

$|\mathcal{U}| = n$ and $P_U(u) = 1/n$ for all $u$, so that $\lim_{n\to\infty} n P_U(u) = 1$. Also, $g(2^n)/P_U(2^n) = (-n/2)/(1/n) = -n^2/2$, so that $\lim_{n\to\infty} \frac{g(2^n)}{P_U(2^n)} = -\infty$.

Exercise 2.6.

A source $X$ taking values in $\{-1,0,1\}$ with equal probability is compressed using the optimal binary prefix-free code. The function $\ell(x)$ returns the length of the encoded version of the symbol $x\in\{-1,0,1\}$. What is the value of Chebyshev's upper bound on the probability $\mathbb{P}[|\ell(X)-L_{\rm av}|\geq L_{\rm av}]$, where $L_{\rm av}$ is the average length of the code?

(a) $0$
(b) $\frac{2}{15}$
(c) $\frac{2}{25}$
(d) $\frac{27}{25}$

Show solution

The correct answer is (c) ★

The probability $\mathbb{P}[|\ell(X)-L_{\rm av}|\geq L_{\rm av}]$ has the form of Chebyshev's inequality since $\mathbb{E}[\ell(X)]=L_{\rm av}$, and is bounded by $\frac{{\rm Var}(\ell(X))}{L_{\rm av}^2}$. After constructing the optimal (Huffman) code for $X$, we find that $\ell(X)\in\{1,2,2\}$ with equal probability, leading to ${\rm Var}(\ell(X)) = \frac{2}{9}$ and $\mathbb{E}[\ell(X)]=\frac{5}{3}$. Hence the bound equals $\frac{2/9}{25/9}=\frac{2}{25}$.

Exercise 2.7.

A source $X$ taking values in $\{-2,-1,1,2\}$ with equal probability is compressed using the optimal binary prefix-free code. For each symbol $x\in\{-2,-1,1,2\}$, the function $\ell(x)$ returns the length of its encoded version. What is the value of the probability $\mathbb{P}\bigl[|\ell(X)-L_{\rm av}|\geq \mathbb{E}[X]\bigr]$, where $L_{\rm av}$ is the average length of the code?

(a) $0$
(b) $\frac{2}{25}$
(c) $1$
(d) Indeterminate

Show solution

The correct answer is (c) ★

The optimal (Huffman) code is given by $\{00,01,10,11\}$, hence $\ell(x)=L_{\rm av}=2$. Since $\mathbb{E}[X]=0$, the event $|\ell(X)-L_{\rm av}|\geq 0$ always holds, so the probability is $1$.

Exercise 2.8.

Let $X$ be a non-negative random variable with $\mathbb{E}[X]=4$. What does Markov's inequality say about $\mathbb{P}[X\geq 8]$?

(a) $\mathbb{P}[X\geq 8]\leq \tfrac{1}{2}$
(b) $\mathbb{P}[X\geq 8]\leq 2$
(c) $\mathbb{P}[X\geq 8]\leq \tfrac{1}{4}$

Show solution

The correct answer is (a) ★

Markov's inequality states $\mathbb{P}[X\geq a]\leq \tfrac{\mathbb{E}[X]}{a}=\tfrac{4}{8}=\tfrac{1}{2}$.

Exercise 2.9.

Let $X$ have mean $\mu=3$ and variance $\sigma^2=1$. What is the Chebyshev bound on $\mathbb{P}[|X-3|\geq 2]$?

(a) $\tfrac{1}{4}$
(b) $\tfrac{1}{2}$
(c) $\tfrac{3}{4}$

Show solution

The correct answer is (a) ★

Chebyshev's inequality gives $\mathbb{P}[|X-\mu|\geq k]\leq \tfrac{\sigma^2}{k^2}=\tfrac{1}{4}$.

Exercise 2.10.

Let $X_1,\ldots,X_n$ be i.i.d. with $\mathbb{E}[X_i]=\mu$ and $\operatorname{var}(X_i)=\sigma^2$. What is $\operatorname{var}\bigl(\tfrac{1}{n}\sum_{i=1}^n X_i\bigr)$?

(a) $\tfrac{\sigma^2}{n}$
(b) $\sigma^2$
(c) $n\sigma^2$

Show solution

The correct answer is (a) ★

By independence, $\operatorname{var}\bigl(\tfrac{1}{n}\sum X_i\bigr)=\tfrac{1}{n^2}\sum \operatorname{var}(X_i)=\tfrac{n\sigma^2}{n^2}=\tfrac{\sigma^2}{n}$.

Exercise 2.11.

A fair die is rolled $n$ times and $Y=\tfrac{1}{n}\sum_{i=1}^n X_i$ is the sample mean. By the law of large numbers, what does $Y$ converge to as $n\to\infty$?

(a) $\tfrac{7}{2}$
(b) $3$
(c) $\tfrac{1}{6}$

Show solution

The correct answer is (a) ★

The law of large numbers states that $Y$ converges in probability to $\mathbb{E}[X_i]=\tfrac{1+2+3+4+5+6}{6}=\tfrac{7}{2}$.

Exercise 2.12.

Let $Z$ be a random variable with $\mathbb{E}[Z]=0$ and $\mathbb{E}[Z^2]=9$. Using Chebyshev's inequality, what is an upper bound on $\mathbb{P}[|Z|\geq 6]$?

(a) $\tfrac{1}{4}$
(b) $\tfrac{1}{2}$
(c) $\tfrac{3}{4}$

Show solution

The correct answer is (a) ★

We have $\operatorname{var}(Z)=\mathbb{E}[Z^2]-(\mathbb{E}[Z])^2=9$. Chebyshev gives $\mathbb{P}[|Z|\geq 6]\leq \tfrac{9}{36}=\tfrac{1}{4}$.

Exercise 2.15. For a random variable $X$ with mean $\mu$, Chebyshev's inequality can be stated as $\mathbb{P}[|X-\mu|\geq k\sigma]\leq$

Exercise 2.17.

A random variable $X$ takes values in $\{-1,0,1\}$ with $P_X(-1)=\frac14$, $P_X(0)=\frac12$, $P_X(1)=\frac14$. Calculate the variance of $X$.

(a) $\mathrm{var}(X)=\tfrac12$
(b) $\mathrm{var}(X)=0$
(c) $\mathrm{var}(X)=\tfrac14$
(d) $\mathrm{var}(X)=1$

Show solution

The correct answer is (a) ★

By symmetry $\mathbb{E}[X]=(-1)\tfrac14+0\cdot\tfrac12+1\cdot\tfrac14=0$. The second moment is $\mathbb{E}[X^2]=1\cdot\tfrac14+0+1\cdot\tfrac14=\tfrac12$. Hence $\mathrm{var}(X)=\mathbb{E}[X^2]-(\mathbb{E}[X])^2=\tfrac12-0=\tfrac12$. (Answer $0$ wrongly uses $\mathbb{E}[X]$; $\tfrac14$ forgets one tail.)

Exercise 2.18.

For $X\in\{-1,0,1\}$ with $P_X(-1)=\frac14$, $P_X(0)=\frac12$, $P_X(1)=\frac14$, let $Y=X^2$. Calculate $\mathrm{var}(Y)$.

(a) $\mathrm{var}(Y)=\tfrac14$
(b) $\mathrm{var}(Y)=\tfrac12$
(c) $\mathrm{var}(Y)=\tfrac18$
(d) $\mathrm{var}(Y)=0$

Show solution

The correct answer is (a) ★

$Y=X^2$ has $P_Y(0)=P_X(0)=\tfrac12$ and $P_Y(1)=P_X(-1)+P_X(1)=\tfrac12$. Then $\mathbb{E}[Y]=\tfrac12$ and $\mathbb{E}[Y^2]=\tfrac12$ (since $Y^2=Y$ on $\{0,1\}$), so $\mathrm{var}(Y)=\tfrac12-(\tfrac12)^2=\tfrac14$. (The value $\tfrac12$ would be $\mathrm{var}(X)$; squaring is nonlinear so the variance must be recomputed.)

Exercise 2.19.

For $X\in\{-1,0,1\}$ with $P_X(-1)=\frac14$, $P_X(0)=\frac12$, $P_X(1)=\frac14$, is $\mathbb{E}[X^2]$ equal to $(\mathbb{E}[X])^2$? Choose the correct statement and reason.

(a) No: $\mathbb{E}[X^2]=\tfrac12>0=(\mathbb{E}[X])^2$, since $\mathbb{E}[X^2]-(\mathbb{E}[X])^2=\mathrm{var}(X)>0$ here
(b) Yes: they are always equal because expectation is linear
(c) No: $\mathbb{E}[X^2]<(\mathbb{E}[X])^2$ because variance is negative here
(d) Yes: they coincide because $\mathbb{E}[X]=0$

Show solution

The correct answer is (a) ★

They are not equal. Rearranging the variance formula gives $\mathbb{E}[X^2]-(\mathbb{E}[X])^2=\mathrm{var}(X)\ge 0$, which is always nonnegative because variance is the expectation of a square. Equality holds only if $\mathrm{var}(X)=0$ (constant $X$). Here $\mathrm{var}(X)=\tfrac12>0$, so $\mathbb{E}[X^2]=\tfrac12>0=(\mathbb{E}[X])^2$, strictly.

Exercise 2.20.

Let $X$ and $Y$ be independent random variables with $\mathbb{E}[X]=\mathbb{E}[Y]=0$ and variances $\sigma_X^2$ and $\sigma_Y^2$. Define $Z=aX^2+bY^2$ with $a,b>0$. For $t>0$, which is a valid upper bound on $\mathbb{P}[Z\geq t]$?

(a) $\dfrac{a\sigma_X^2+b\sigma_Y^2}{t}$
(b) $\dfrac{a\sigma_X^2+b\sigma_Y^2}{t^2}$
(c) $\dfrac{a^2\sigma_X^2+b^2\sigma_Y^2}{t}$
(d) $\dfrac{\sigma_X^2+\sigma_Y^2}{t}$

Show solution

The correct answer is (a) ★

Since $a,b>0$ and $X^2,Y^2\geq 0$, $Z\geq 0$, so Markov's inequality applies: $\mathbb{P}[Z\geq t]\leq \mathbb{E}[Z]/t$. By linearity, $\mathbb{E}[Z]=a\mathbb{E}[X^2]+b\mathbb{E}[Y^2]$, and since the means are zero, $\mathbb{E}[X^2]=\sigma_X^2$ and $\mathbb{E}[Y^2]=\sigma_Y^2$. Hence $\mathbb{E}[Z]=a\sigma_X^2+b\sigma_Y^2$ and the bound is $\dfrac{a\sigma_X^2+b\sigma_Y^2}{t}$.

Exercise 2.21.

A discrete random variable $X$ takes the values $\{-\sqrt{2},\,+\sqrt{2}\}$, each with probability $\tfrac{1}{2}$. For $\epsilon>0$, what is Chebyshev's upper bound on $\mathbb{P}\big[\,|X^2-2|\geq\epsilon\,\big]$?

(a) $0$
(b) $\dfrac{2}{\epsilon^2}$
(c) $\dfrac{1}{\epsilon^2}$
(d) $\dfrac{1}{2}$

Show solution

The correct answer is (a) ★

Let $Y=X^2$. Since $X=\pm\sqrt{2}$, we get $Y=(\pm\sqrt{2})^2=2$ with probability one, so $X^2$ is the constant $2$. Then $\mathbb{E}[X^2]=2$ and $\mathrm{var}(X^2)=0$. The event is exactly the deviation of $X^2$ from its mean, so Chebyshev gives $\mathbb{P}[\,|X^2-2|\geq\epsilon\,]\leq \mathrm{var}(X^2)/\epsilon^2 = 0/\epsilon^2 = \boxed{0}$.

Exercise 2.22.

Let $X_1$ take values in $\{0,1,2\}$ with $P_X(0)=\tfrac14$, $P_X(1)=\tfrac12$, $P_X(2)=\tfrac14$, so $\mathbb{E}[X_1]=1$. Using Markov's inequality, calculate the upper bound on $\mathbb{P}[X_1\geq 2]$.

(a) $\tfrac12$
(b) $\tfrac14$
(c) $1$
(d) $\tfrac18$

Show solution

The correct answer is (a) ★

Markov's inequality for a non-negative variable gives $\mathbb{P}[X_1\geq a]\leq \mathbb{E}[X_1]/a$. With $a=2$ and $\mathbb{E}[X_1]=1$, the bound is $1/2$. (The exact value $P_X(2)=\tfrac14$ is smaller, so the bound is valid but loose.) The answer is $\tfrac12$.

Exercise 2.23.

Let $X_1,\ldots,X_n$ be i.i.d. with $\mathrm{var}(X_i)=\tfrac12$, and let $\bar X_n=\tfrac1n\sum_{i=1}^n X_i$ be the sample mean. Calculate $\mathrm{var}(\bar X_n)$.

(a) $\dfrac{1}{2n}$
(b) $\dfrac{1}{2n^2}$
(c) $\dfrac{1}{2}$
(d) $\dfrac{n}{2}$

Show solution

The correct answer is (a) ★

By independence, $\mathrm{var}(\bar X_n)=\tfrac{1}{n^2}\sum_{i=1}^n\mathrm{var}(X_i)=\tfrac{1}{n^2}\cdot n\cdot\tfrac12=\tfrac{1}{2n}$. The $1/n^2$ factor combines with the $n$ terms to leave a single $1/n$. The answer is $\tfrac{1}{2n}$.

Exercise 2.24.

With $\bar X_n$ the sample mean of i.i.d. variables, $\mathbb{E}[\bar X_n]=1$ and $\mathrm{var}(\bar X_n)=\tfrac{1}{2n}$. Using Chebyshev's inequality, calculate the minimum number of samples $n$ such that $\mathbb{P}\bigl[\,|\bar X_n-1|<\tfrac12\,\bigr]\geq 0.9$.

(a) $n=20$
(b) $n=10$
(c) $n=5$
(d) $n=40$

Show solution

The correct answer is (a) ★

The condition $\mathbb{P}[|\bar X_n-1|<\tfrac12]\geq 0.9$ is equivalent to $\mathbb{P}[|\bar X_n-1|\geq\tfrac12]\leq 0.1$. Chebyshev gives $\mathbb{P}[|\bar X_n-1|\geq\tfrac12]\leq \dfrac{\mathrm{var}(\bar X_n)}{(1/2)^2}=\dfrac{1/(2n)}{1/4}=\dfrac{2}{n}$. Setting $\tfrac{2}{n}\leq 0.1$ gives $n\geq 20$. (Dividing by $k=\tfrac12$ instead of $k^2=\tfrac14$ wrongly yields $10$.) The answer is $n=20$.

Exercise 2.25.

Let $X$ be uniform on $\{1,2,\ldots,M\}$, so $P_X(x)=\tfrac{1}{M}$. Define $Y=f(X)$ with $f(x)=P_X(x)$. Compute $\mathrm{var}(Y)$.

(a) $0$
(b) $\tfrac{1}{M}$
(c) $\tfrac{1}{M^2}$
(d) $\tfrac{M^2-1}{12}$

Show solution

The correct answer is (a) ★

Since $f(x)=P_X(x)=\tfrac1M$ does not depend on $x$, $Y\equiv\tfrac1M$ is a constant (degenerate) random variable. Then $\mathbb{E}[Y]=\tfrac1M$ and $\mathbb{E}[Y^2]=\tfrac1{M^2}$, so $\mathrm{var}(Y)=\tfrac1{M^2}-(\tfrac1M)^2=0$.

Exercise 2.26.

Let $X$ be uniform on $\{1,2,\ldots,M\}$. Define $Y=f(X)$ with $f(x)=x^2$. Using $\sum_{n=1}^{M} n^2=\tfrac{M(M+1)(2M+1)}{6}$, compute $\mathbb{E}[Y]$.

(a) $\dfrac{(M+1)(2M+1)}{6}$
(b) $\dfrac{M(M+1)(2M+1)}{6}$
(c) $\dfrac{(M+1)^2}{4}$
(d) $\dfrac{M+1}{2}$

Show solution

The correct answer is (a) ★

By the law of the unconscious statistician, $\mathbb{E}[Y]=\mathbb{E}[X^2]=\tfrac1M\sum_{n=1}^{M} n^2=\tfrac1M\cdot\tfrac{M(M+1)(2M+1)}{6}=\tfrac{(M+1)(2M+1)}{6}$. The second option forgets the $\tfrac1M$ averaging factor; the last is $\mathbb{E}[X]$, not $\mathbb{E}[X^2]$.

Exercise 2.27.

Let $X$ be uniform on $\{1,2,3,4\}$ (so $P_X(x)=\tfrac14$) and let $Y=f(X)$ with $f(x)=|x-2|$. Define $Z=\log_2 P_Y(Y)$. Compute $\mathbb{E}[Z]$ (in bits).

(a) $-\tfrac32$
(b) $\tfrac32$
(c) $-2$
(d) $-1$

Show solution

The correct answer is (a) ★

Mapping $|x-2|$ gives $Y\in\{0,1,2\}$ with $P_Y(0)=\tfrac14$, $P_Y(1)=\tfrac14+\tfrac14=\tfrac12$, $P_Y(2)=\tfrac14$. Then $\mathbb{E}[Z]=\sum_y P_Y(y)\log_2 P_Y(y)=\tfrac14(-2)+\tfrac12(-1)+\tfrac14(-2)=-\tfrac32$ bits. Note $-\mathbb{E}[Z]=\tfrac32=H(Y)$; option $+\tfrac32$ drops the sign.

Exercise 2.28.

Let $X_1,X_2,X_3,X_4$ be i.i.d. Bernoulli$(\tfrac12)$ random variables in $\{0,1\}$, and let $Y=\tfrac14(X_1+X_2+X_3+X_4)$. Which of the following is the correct probability mass function of $Y$?

(a) $\mathbb{P}[Y=0]=\tfrac{1}{16},\ \mathbb{P}[Y=\tfrac14]=\tfrac{4}{16},\ \mathbb{P}[Y=\tfrac12]=\tfrac{6}{16},\ \mathbb{P}[Y=\tfrac34]=\tfrac{4}{16},\ \mathbb{P}[Y=1]=\tfrac{1}{16}$
(b) $\mathbb{P}[Y=0]=\tfrac{1}{5},\ \mathbb{P}[Y=\tfrac14]=\tfrac{1}{5},\ \mathbb{P}[Y=\tfrac12]=\tfrac{1}{5},\ \mathbb{P}[Y=\tfrac34]=\tfrac{1}{5},\ \mathbb{P}[Y=1]=\tfrac{1}{5}$
(c) $\mathbb{P}[Y=0]=\tfrac{4}{16},\ \mathbb{P}[Y=\tfrac14]=\tfrac{1}{16},\ \mathbb{P}[Y=\tfrac12]=\tfrac{6}{16},\ \mathbb{P}[Y=\tfrac34]=\tfrac{1}{16},\ \mathbb{P}[Y=1]=\tfrac{4}{16}$
(d) $\mathbb{P}[Y=0]=\tfrac{1}{16},\ \mathbb{P}[Y=\tfrac14]=\tfrac{2}{16},\ \mathbb{P}[Y=\tfrac12]=\tfrac{10}{16},\ \mathbb{P}[Y=\tfrac34]=\tfrac{2}{16},\ \mathbb{P}[Y=1]=\tfrac{1}{16}$

Show solution

The correct answer is (a) ★

Write $S=X_1+\cdots+X_4=4Y$. The $2^4=16$ binary strings are equally likely, each with probability $\tfrac{1}{16}$, and $S$ counts the ones. The number of strings with $k$ ones is $\binom{4}{k}=1,4,6,4,1$ for $k=0,1,2,3,4$. Dividing by $16$ gives $\mathbb{P}[Y=\tfrac{k}{4}]=\tfrac{1}{16},\tfrac{4}{16},\tfrac{6}{16},\tfrac{4}{16},\tfrac{1}{16}$.

Exercise 2.29.

With $Y=\tfrac14(X_1+X_2+X_3+X_4)$ for i.i.d. Bernoulli$(\tfrac12)$ variables (so $\mathbb{P}[Y=\tfrac{k}{4}]=\binom{4}{k}/16$ and $\mathbb{E}[Y]=\tfrac12$), compute the exact probability $\mathbb{P}[Y\geq\tfrac34]$ and the Markov upper bound on it. Which pair is correct?

(a) exact $=\tfrac{5}{16}\approx0.3125$, Markov bound $=\tfrac{2}{3}\approx0.667$
(b) exact $=\tfrac{1}{16}\approx0.0625$, Markov bound $=\tfrac{2}{3}\approx0.667$
(c) exact $=\tfrac{5}{16}\approx0.3125$, Markov bound $=\tfrac{3}{8}=0.375$
(d) exact $=\tfrac{4}{16}=0.25$, Markov bound $=\tfrac{3}{2}=1.5$

Show solution

The correct answer is (a) ★

The values $Y\geq\tfrac34$ are $Y=\tfrac34$ and $Y=1$, so $\mathbb{P}[Y\geq\tfrac34]=\tfrac{4}{16}+\tfrac{1}{16}=\tfrac{5}{16}=0.3125$. Markov's inequality for the non-negative $Y$ gives $\mathbb{P}[Y\geq a]\leq\mathbb{E}[Y]/a$; with $\mathbb{E}[Y]=\tfrac12$ and $a=\tfrac34$ the bound is $\tfrac{1/2}{3/4}=\tfrac{2}{3}\approx0.667$. The exact value lies well below this loose bound.

Exercise 2.30.

With $Y=\tfrac14(X_1+X_2+X_3+X_4)$ for i.i.d. Bernoulli$(\tfrac12)$ variables, we have $\mathbb{E}[Y]=\tfrac12$ and $\mathrm{var}(Y)=\tfrac{1}{16}$. Compute the exact probability $\mathbb{P}[\,|Y-\tfrac12|\geq\tfrac12\,]$ and the Chebyshev upper bound on it. Which pair is correct?

Show solution

The correct answer is (a) ★

The event $|Y-\tfrac12|\geq\tfrac12$ is realized only by $Y=0$ and $Y=1$, so the exact probability is $\tfrac{1}{16}+\tfrac{1}{16}=\tfrac{2}{16}=\tfrac18=0.125$. Chebyshev's inequality gives $\mathbb{P}[|Y-\mathbb{E}[Y]|\geq\varepsilon]\leq\mathrm{var}(Y)/\varepsilon^2$; with $\mathrm{var}(Y)=\tfrac{1}{16}$ and $\varepsilon=\tfrac12$ the bound is $\tfrac{1/16}{1/4}=\tfrac14=0.25$. The exact value $0.125$ satisfies this bound.

Exercise 2.31.

Let $V$ be an i.i.d. equiprobable binary source and $\mathbf{V}=(V_1,\ldots,V_n)$ a length-$n$ sequence. Define $g(\mathbf{v})=\log_2 P_{\mathbf{V}}(\mathbf{v})$. Find the Markov upper bound on $\mathbb{P}\bigl[g(\mathbf{V})\leq -n/2\bigr]$.

(a) $2$
(b) $1$
(c) $1/2$
(d) $1/n$

Show solution

The correct answer is (a) ★

Each sequence has probability $2^{-n}$, so $g(\mathbf{V})=-n$ always, giving $\mathbb{E}[g]=-n$. Set $X=-g\geq 0$ with $\mathbb{E}[X]=n$; the event $g\leq -n/2$ is $X\geq n/2$. Markov gives $\mathbb{P}[X\geq n/2]\leq \mathbb{E}[X]/(n/2)=n/(n/2)=2$. The bound is $2$ (vacuous, since it exceeds $1$).

Exercise 2.32.

For an i.i.d. equiprobable binary source, $\mathbf{V}=(V_1,\ldots,V_n)$ and $g(\mathbf{v})=\log_2 P_{\mathbf{V}}(\mathbf{v})$. Compute the exact value of $\mathbb{P}\bigl[g(\mathbf{V})\leq -n/2\bigr]$.

(a) $1$
(b) $2$
(c) $1/2$
(d) $2^{-n/2}$

Show solution

The correct answer is (a) ★

Since every sequence has probability $2^{-n}$, $g(\mathbf{V})=-n$ deterministically. For all $n\geq 1$, $-n\leq -n/2$, so the event $\{g\leq -n/2\}$ is certain: $\mathbb{P}=1$. This is consistent with the Markov bound $2$ ($1\leq 2$) but much tighter, showing Markov is loose here.

Exercise 2.33.

For an i.i.d. equiprobable binary source, $g(\mathbf{v})=\log_2 P_{\mathbf{V}}(\mathbf{v})$ with $\mathbf{V}=(V_1,\ldots,V_n)$. Compute $\mathbb{P}\bigl[\,|g(\mathbf{V})-\mathbb{E}[g(\mathbf{V})]|\geq 0\,\bigr]$.

(a) $1$
(b) $0$
(c) $1/2$
(d) $n$

Show solution

The correct answer is (a) ★

An absolute value is always $\geq 0$, so this is the sure event regardless of the distribution: $\mathbb{P}=1$. Concretely $g(\mathbf{V})=-n=\mathbb{E}[g]$, so the deviation is exactly $0$ and $|0|\geq 0$ holds with certainty.

Exercise 2.34.

For an i.i.d. equiprobable binary source, $g(\mathbf{v})=\log_2 P_{\mathbf{V}}(\mathbf{v})$ with $\mathbf{V}=(V_1,\ldots,V_n)$. Find the Chebyshev upper bound on $\mathbb{P}\bigl[\,|g(\mathbf{V})-\mathbb{E}[g(\mathbf{V})]|\geq\epsilon\,\bigr]$ for $\epsilon>0$.

(a) $0$
(b) $1$
(c) $n/\epsilon^2$
(d) $1/\epsilon^2$

Show solution

The correct answer is (a) ★

Chebyshev: $\mathbb{P}[|g-\mathbb{E}[g]|\geq\epsilon]\leq \mathrm{var}(g)/\epsilon^2$. Since $g(\mathbf{V})=-n$ is deterministic, $\mathrm{var}(g)=0$, so the bound is $0/\epsilon^2=0$ for every $\epsilon>0$. It is tight: a deterministic variable never deviates from its mean.

Exercise 2.35.

Let $U$ be uniform on $\mathcal{U}=\{2^1,2^2,\ldots,2^n\}$ and let $g(u)=-\log_2\!\big(\sqrt{u}\big)$. Using $\sum_{k=1}^{n}k=\frac{n(n+1)}{2}$, what is $\mathbb{E}[g(U)]$?

(a) $-\dfrac{n+1}{4}$
(b) $-\dfrac{n+1}{2}$
(c) $-\dfrac{n(n+1)}{2}$
(d) $\dfrac{n+1}{4}$

Show solution

The correct answer is (a) ★

Index outcomes as $u=2^a$ with $a\in\{1,\ldots,n\}$, each with probability $\tfrac1n$. By the log power rule, $g(2^a)=-\tfrac12\log_2(2^a)=-\tfrac a2$. Then $\mathbb{E}[g(U)]=-\tfrac{1}{2n}\sum_{a=1}^n a=-\tfrac{1}{2n}\cdot\tfrac{n(n+1)}{2}=-\tfrac{n+1}{4}$.

Exercise 2.36.

Let $U$ be uniform on $\mathcal{U}=\{2^1,2^2,\ldots,2^n\}$ and let $g(u)=-\log_2\!\big(\sqrt{u}\big)$. Using $\sum_{k=1}^{n}k^2=\frac{n(n+1)(2n+1)}{6}$, what is $\mathbb{E}[g^2(U)]$?

(a) $\dfrac{(n+1)(2n+1)}{24}$
(b) $\dfrac{(n+1)(2n+1)}{6}$
(c) $\dfrac{(n+1)(2n+1)}{12}$
(d) $\dfrac{(n+1)^2}{16}$

Show solution

The correct answer is (a) ★

With $g(2^a)=-\tfrac a2$, we have $g^2(2^a)=\tfrac{a^2}{4}$. By LOTUS, $\mathbb{E}[g^2(U)]=\tfrac{1}{4n}\sum_{a=1}^n a^2=\tfrac{1}{4n}\cdot\tfrac{n(n+1)(2n+1)}{6}=\tfrac{(n+1)(2n+1)}{24}$.

Exercise 2.37.

Let $U$ be uniform on $\mathcal{U}=\{2^1,2^2,\ldots,2^n\}$ and let $g(u)=-\log_2\!\big(\sqrt{u}\big)$. Given $\mathbb{E}[g(U)]=-\tfrac{n+1}{4}$ and $\mathbb{E}[g^2(U)]=\tfrac{(n+1)(2n+1)}{24}$, for which value of $n$ is $\mathrm{var}(g(U))=1$?

(a) $n=7$
(b) $n=49$
(c) $n=5$
(d) $n=8$

Show solution

The correct answer is (a) ★

Using $\mathrm{var}=\mathbb{E}[g^2]-(\mathbb{E}[g])^2=\tfrac{(n+1)(2n+1)}{24}-\tfrac{(n+1)^2}{16}$, factor $\tfrac{n+1}{48}$ to get $\mathrm{var}=\tfrac{(n+1)(n-1)}{48}=\tfrac{n^2-1}{48}$. Setting this to $1$: $n^2-1=48$, so $n^2=49$ and (positive integer root) $n=7$.

Exercise 2.38.

Let $(X_1,\ldots,X_n)$ be i.i.d. uniform on $\{1,2,\ldots,M\}$ and let $\overline{X}=\frac{X_1+\cdots+X_n}{\sqrt{n}}$. Using Markov's inequality, which bound holds for $\mathbb{P}[\overline{X}\geq n]$?

(a) $\mathbb{P}[\overline{X}\geq n]\leq \dfrac{M+1}{2\sqrt{n}}$
(b) $\mathbb{P}[\overline{X}\geq n]\leq \dfrac{M+1}{2n}$
(c) $\mathbb{P}[\overline{X}\geq n]\leq \dfrac{M+1}{2\sqrt{n}}\cdot n=\dfrac{(M+1)\sqrt{n}}{2}$
(d) $\mathbb{P}[\overline{X}\geq n]\leq \dfrac{M-1}{2\sqrt{n}}$

Show solution

The correct answer is (a) ★

Since each $X_i\geq 1$, $\overline{X}$ is non-negative, so Markov gives $\mathbb{P}[\overline{X}\geq a]\leq \mathbb{E}[\overline{X}]/a$. By linearity $\mathbb{E}[\overline{X}]=\frac{n\,\mathbb{E}[X]}{\sqrt{n}}=\sqrt{n}\cdot\frac{M+1}{2}$ since $\mathbb{E}[X]=\frac{M+1}{2}$. With $a=n$: $\mathbb{P}[\overline{X}\geq n]\leq \frac{\sqrt{n}(M+1)/2}{n}=\frac{M+1}{2\sqrt{n}}$.

Exercise 2.39.

Let $(X_1,\ldots,X_n)$ be i.i.d. uniform on $\{1,2,\ldots,M\}$ (so $\mathrm{var}(X)=\frac{M^2-1}{12}$) and let $\overline{X}=\frac{X_1+\cdots+X_n}{\sqrt{n}}$. What is $\mathrm{var}(\overline{X})$?

(a) $\mathrm{var}(\overline{X})=\dfrac{M^2-1}{12}$
(b) $\mathrm{var}(\overline{X})=\dfrac{M^2-1}{12n}$
(c) $\mathrm{var}(\overline{X})=\dfrac{M^2-1}{12\sqrt{n}}$
(d) $\mathrm{var}(\overline{X})=\dfrac{n(M^2-1)}{12}$

Show solution

The correct answer is (a) ★

Using the scaling rule $\mathrm{var}(cY)=c^2\mathrm{var}(Y)$ with $c=1/\sqrt{n}$ and independence: $\mathrm{var}(\overline{X})=\frac{1}{n}\sum_{i=1}^n\mathrm{var}(X_i)=\frac{1}{n}\cdot n\,\mathrm{var}(X)=\mathrm{var}(X)=\frac{M^2-1}{12}$. The $\sqrt{n}$ scaling exactly cancels the factor $n$, so the variance is independent of $n$ (the CLT normalization).

Exercise 2.40.

Let $(X_1,\ldots,X_n)$ be i.i.d. uniform on $\{1,2,\ldots,M\}$ with $\overline{X}=\frac{X_1+\cdots+X_n}{\sqrt{n}}$, so $\mathrm{var}(\overline{X})=\frac{M^2-1}{12}$ (constant in $n$). For $M\geq 2$ and $\epsilon>0$, using Chebyshev's inequality, which deviation probability is guaranteed to converge to $0$ as $n\to\infty$?

(a) $\mathbb{P}\big[|\overline{X}-\mathbb{E}[\overline{X}]|\geq \sqrt{n\epsilon}\big]$
(b) $\mathbb{P}\big[|\overline{X}-\mathbb{E}[\overline{X}]|\geq \tfrac{1}{\sqrt{n\epsilon}}\big]$
(c) $\mathbb{P}\big[|\overline{X}-\mathbb{E}[\overline{X}]|\geq \epsilon\big]$
(d) All three converge to $0$

Show solution

The correct answer is (a) ★

Chebyshev gives $\mathbb{P}[|\overline{X}-\mathbb{E}[\overline{X}]|\geq t]\leq \frac{M^2-1}{12\,t^2}$. For $t=\sqrt{n\epsilon}$, $t^2=n\epsilon$ and the bound is $\frac{M^2-1}{12n\epsilon}\to 0$. For $t=\frac{1}{\sqrt{n\epsilon}}$ the bound grows like $n$, and for fixed $t=\epsilon$ the bound is a constant; neither shrinks. Since $\mathrm{var}(\overline{X})$ does not decrease with $n$, only the growing threshold $\sqrt{n\epsilon}$ guarantees convergence to $0$.
