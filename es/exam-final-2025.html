Examen final 2025

Problema 1. Un generador de palabras aleatorias emite palabras de $r$ letras, donde cada letra pertenece a un alfabeto de tamaño $n$. ¿Cuál es la entropía binaria máxima del generador de palabras aleatorias?

(a) $\log_2(r)$
(b) $\log_2(n)$
(c) $r\log_2(n)$
(d) $n\log_2(r)$

Mostrar solución

La respuesta correcta es (c) ★

El generador de palabras aleatorias se puede describir como una fuente discreta $X=(U_1,\ldots,U_r)$ donde cada $U_i$ toma valores en un alfabeto de cardinalidad $n$. La entropía de $X$ satisface $H(X)\leq\log_2|\mathcal{X}|$, donde $|\mathcal{X}|=n^r$.

Problema 2. Un código fuente binario generado aleatoriamente con palabras código $\{\boldsymbol{x}_1,\boldsymbol{x}_2\}$, tal que cada palabra código se selecciona independientemente del conjunto $\{0,01,10\}$ con distribución de probabilidad $\bigl\{\tfrac12,\tfrac13,\tfrac16\bigr\}$, se utiliza para representar una fuente discreta sin memoria que toma valores en $\{a,b\}$ con distribución de probabilidad $\bigl\{\tfrac13,\tfrac23\bigr\}$. ¿Cuál es la probabilidad de generar un código unívocamente decodificable con longitud media $\tfrac53$?

(a) $0$
(b) $\tfrac14$
(c) $\tfrac19$
(d) $\tfrac{1}{12}$

Mostrar solución

La respuesta correcta es (b) ★

De todos los posibles códigos generados aleatoriamente, solo $\{0,01\}$ y $\{0,10\}$ satisfacen ambas condiciones de decodificabilidad unívoca y longitud media. La probabilidad de dichos códigos es $\tfrac 16$ y $\tfrac{1}{12}$, respectivamente, cuya suma lleva a la respuesta correcta.

Problema 3. Considere una fuente $W_n$ que emite secuencias de $n$ bits i.i.d. tal que la probabilidad de $1$ es el doble de la probabilidad de $0$. La longitud media del código binario óptimo libre de prefijos para representar $W_n$ se denota como $L_{\rm av}(W_n)$. ¿Cuál es el valor del siguiente cociente?$$\frac{L_{\rm av}(W_1)}{L_{\rm av}(W_2)}$$

(a) $1$
(b) $\tfrac{9}{23}$
(c) $\tfrac{9}{17}$
(d) $\tfrac12$

Mostrar solución

La respuesta correcta es (c) ★

Claramente $L_{\rm av}(W_1)=1$. Usando que cada bit toma valores $\{0,1\}$ con distribución de probabilidad $\bigl\{\tfrac13,\tfrac23\bigr\}$, obtenemos que $W_2\in\{00,01,10,11\}$ con probabilidades $\bigl\{\tfrac19,\tfrac29,\tfrac29,\tfrac49\bigr\}$. El código de Huffman para $W_2$ tiene distribución de longitudes $\{3,3,2,1\}$ dando $L_{\rm av}(W_2)=\tfrac{17}{9}$.

Problema 4. Una fuente discreta sin memoria $W_n=(X_1,\ldots,X_n)$, donde cada $X_i$ toma valores en $\{a,b,c\}$ con distribución de probabilidad $\bigl\{\tfrac12,\tfrac13,\tfrac16\}$, se representa usando dos esquemas de codificación. En el esquema (a), diseñamos el código binario óptimo libre de prefijos para $X_1$ y representamos cada símbolo $X_i$ con el mismo código. En el esquema (b), diseñamos el código binario óptimo libre de prefijos para $W_n$. La longitud media para representar $W_n$ usando los esquemas (a) y (b) se denota respectivamente como $L_{\rm av}^\text{(a)}(W_n)$ y $L_{\rm av}^\text{(b)}(W_n)$. ¿Cuál es el valor del siguiente límite?$$\lim_{n\to\infty} \frac{L_{\rm av}^\text{(b)}(W_n)}{L_{\rm av}^\text{(a)}(W_n)}$$

(a) $\tfrac{4+3\log_2(3)}{10}$
(b) $\tfrac{4+3\log_2(3)}{9}$
(c) $\tfrac{4+3\log_2(3)}{6}$
(d) $\tfrac{4+3\log_2(3)}{11}$

Mostrar solución

La respuesta correcta es (b) ★

El código de Huffman para $X_1$ tiene distribución de longitudes $\{1,2,2\}$ y longitud media $L_{\rm av}(X_1)=\tfrac32$. En el esquema (a), $L_{\rm av}(W_n) = \tfrac{3n}{2}$ al concatenar el mismo código. Por otro lado, cuando $n\to\infty$, $\frac1n L_{\rm av}(W_n) = H(X_1)$. Combinando ambos hechos y usando $H(X_1)= \tfrac 23 + \tfrac{\log_2(3)}{2}$, obtenemos la respuesta correcta.

Problema 5. Las palabras código de un código lineal binario con matriz generadora$$\begin{pmatrix}0 & 1\\1 & 0\end{pmatrix}$$se transmiten con igual probabilidad por un canal binario con probabilidades condicionales $P_{Y|X}(0|0)=0$ y $P_{Y|X}(0|1)=\tfrac13$. El receptor usa una regla de distancia de Hamming máxima para decodificar la salida del canal. ¿Cuál es la probabilidad de error del código?

(a) $\tfrac{11}{36}$
(b) $\tfrac{5}{9}$
(c) $\tfrac{8}{9}$
(d) $\tfrac{35}{36}$

Mostrar solución

La respuesta correcta es (b) ★

Dado que el código generado es $\{00,10,01,11\}$ y el receptor usa una regla de distancia de Hamming máxima, el decodificador $\hat m(\boldsymbol{y})$ es tal que $\hat m(00)=4$, $\hat m(01)=2$, $\hat m(10)=3$ y $\hat m(11)=1$. Por lo tanto, $P_{\rm e}(1)=0$, $P_{\rm e}(2)=\tfrac 23$, $P_{\rm e}(3)=\tfrac 23$ y $P_{\rm e}(4)=\tfrac89$. Usando que las cuatro palabras código son equiprobables, obtenemos el resultado.

Problema 6. Un canal binario tiene entrada $X$ que toma valores en $\{0,1\}$ con igual probabilidad, salida $Y$ que toma valores en $\{0,\star,1\}$, y probabilidades condicionales $P_{Y|X}(y|x)=\tfrac13$ para todo $x$ e $y$. ¿Cuál es el valor de la entropía condicional $H(X|Y)$?

(a) $1$
(b) $\tfrac{3\log_2(3)-1}{3}$
(c) $\tfrac{3\log_2(3)-2}{3}$
(d) $0$

Mostrar solución

La respuesta correcta es (a) ★

Como $H(Y|X)=H(Y)=\log_2(3)$, la información mutua del canal satisface $I(X;Y)=0$, lo que también implica que $H(X|Y)=H(X)=1$.

Problema 7. Tres variables aleatorias $X_1$, $X_2$ y $X_3$, forman una cadena de Markov $X_1\rightarrow X_2\rightarrow X_3$, donde además $X_2$ es una función determinista de $X_1$. ¿Cuál es la entropía conjunta de las tres variables aleatorias?

(a) $H(X_1)+H(X_2)+H(X_3)$
(b) $H(X_1)+H(X_3)$
(c) $H(X_2)+H(X_3|X_2)$
(d) $H(X_1)+H(X_3|X_2)$

Mostrar solución

La respuesta correcta es (d) ★

La entropía conjunta se puede expandir como $H(X_1)+H(X_2|X_1)+H(X_3|X_1,X_2)$. Como $X_2$ es una función determinista de $X_1$, tenemos $H(X_2|X_1)=0$. Además, condicionado a $X_2$, $X_1$ y $X_3$ se vuelven independientes, así que $H(X_3|X_1,X_2)=H(X_3|X_2)$. Combinando ambas observaciones, obtenemos la respuesta correcta.
