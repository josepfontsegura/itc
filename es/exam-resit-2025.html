Examen de recuperación 2025

Problema 1. Un experimento consta de dos pasos. Primero, se lanza una moneda justa y se registra el resultado (cara o cruz). Si el resultado es cara, se lanza un dado justo de seis caras y se registra su cara; si el resultado es cruz, se lanza una segunda moneda justa y se registra su resultado. ¿Cuál es la entropía en bits del resultado del experimento de dos pasos?

(a) $\tfrac{1+\log_2(6)}{2}$
(b) $\tfrac{3+\log_2(3)}{2}$
(c) $\tfrac{4+\log_2(3)}{2}$
(d) $\tfrac{2+\log_2(6)}{2}$

Mostrar solución

La respuesta correcta es (c) ★

El resultado se puede modelar como $X=(A,B)$. Después de determinar todos los resultados, $X$ tiene distribución $\bigl\{\tfrac{1}{12}, \tfrac{1}{12}, \tfrac{1}{12}, \tfrac{1}{12}, \tfrac{1}{12}, \tfrac{1}{12}, \tfrac{1}{4}, \tfrac{1}{4}\bigr\}$. Usando la regla de la cadena $H(X)=H(A)+H(B|A)$, con $H(A)=1$ y $H(B|A)=\tfrac{1+\log_2(6)}{2}$.

Problema 2. Una fuente discreta toma valores en $\{a,b,c\}$ con probabilidades $\{p,2p,3p\}$ para algún parámetro $p\neq 0$. ¿Cuál es la longitud media del código binario óptimo libre de prefijos para esta fuente?

(a) $\log_2(3)$
(b) $\tfrac 32$
(c) $2$
(d) $\tfrac{4+3\log_2(3)}{6}$

Mostrar solución

La respuesta correcta es (b) ★

Solo $p=\tfrac 16$ lleva a una distribución de probabilidad válida. El código de Huffman para la fuente tiene longitudes $\{2,2,1\}$, por lo que $L_{\rm av}=9p=\tfrac 32$.

Problema 3. Una fuente discreta sin memoria $W_n=(X_1,\ldots,X_n)$, donde cada $X_i\in\{a,b,c,d,e,f\}$ con igual probabilidad, se codifica usando el código binario óptimo libre de prefijos. ¿En cuál de los siguientes intervalos se encuentra la longitud media por símbolo del código binario óptimo libre de prefijos para $W_{3}$?

(a) $\bigl[\tfrac{3+3\log_2(3)}{2},\tfrac{3\log_2(3) + 4}{2} \bigr)$
(b) $\bigl[\tfrac{1+\log_2(3)}{2},\tfrac{3\log_2(3) + 4}{6} \bigr)$
(c) $\bigl[1+\log_2(3),\tfrac{3\log_2(3) + 4}{3} \bigr)$
(d) $\bigl[3+3\log_2(3),3\log_2(3) + 4 \bigr)$

Mostrar solución

La respuesta correcta es (c) ★

La longitud media por símbolo se encuentra en $\bigl[H(X_1),H(X_1)+\tfrac 13\bigr)$. Usando $H(X_1)=\log_2(6)$, obtenemos el resultado.

Problema 4. Defina la variable aleatoria $Z$ como $Z=\log_2 P_X(X)$, donde $X$ es una fuente discreta sin memoria $X=(U_1,\ldots,U_r)$, con cada $U_i$ tomando valores en $\{1,2,3,\ldots,n\}$ con igual probabilidad. ¿Cuál es el valor esperado de $Z$?

(a) $r\log_2(n)$
(b) $-r\log_2(n)$
(c) $n\log_2(r)$
(d) $-n\log_2(r)$

Mostrar solución

La respuesta correcta es (b) ★

Como $P_X(x)=\tfrac{1}{n^r}$ para todo $x$, la variable aleatoria es determinista con valor $\log_2(n^{-r})$.

Problema 5. Las palabras código de un cierto código lineal $(5,1)$ con distribución de pesos que satisface $A_2=1$ se transmiten con igual probabilidad por un canal binario simétrico con probabilidad de cambio de bit $p$. ¿Cuál es la probabilidad de recibir la secuencia de todos unos?

(a) $\tfrac{p^5-2p^4 +2p^3}{2}$
(b) $\tfrac{2p^5 -2p^4 +p^3}{2}$
(c) $\tfrac{p^5-p^4+p^3}{2}$
(d) $\tfrac{p^5+p^4+p^3}{2}$

Mostrar solución

La respuesta correcta es (b) ★

El código tiene dos palabras código, $00000$ y una segunda palabra código con $2$ unos y $3$ ceros. La probabilidad de recibir $11111$ es $\tfrac 12 p^5 + \tfrac 12 p^3(1-p)^2$. Expandiendo y simplificando, obtenemos el resultado.

Problema 6. El código $\{000,111\}$ se transmite por un canal binario con probabilidades condicionales $P_{Y|X}(0|0)=1$ y $P_{Y|X}(0|1)=\tfrac 12$. El receptor usa una regla de decodificación de mínima verosimilitud, es decir $\hat m = \arg\min_m P_{\boldsymbol{Y}|\boldsymbol{X}}(\boldsymbol{y}|\boldsymbol{x}_m)$, para decodificar la salida del canal. ¿Cuál es la región de error del segundo mensaje?

(a) $\{001,010,011,100,101,110,111\}$
(b) $\{000\}$
(c) $\{000,001,010,100\}$
(d) $\{011,101,110,111\}$

Mostrar solución

La respuesta correcta es (a) ★

Como $P_{Y|X}(1|0)=0$, tenemos $P_{\boldsymbol{Y}|\boldsymbol{X}}(\boldsymbol{y}|000)=0$ para cualquier salida con al menos un $1$. Por lo tanto, $\hat m = 1$ para todas las salidas excepto $\boldsymbol{y}=000$, para la cual $\hat m = 2$.

Problema 7. Dos variables aleatorias $X$ e $Y$ toman valores en $\{a,b\}$ y $\{a,b,c,d\}$, respectivamente, de modo que $X$ tiene valores equiprobables y $H(Y|X)=\tfrac 32$. ¿Cuál de las siguientes relaciones es falsa?

(a) $H(X,Y) = 3$
(b) $H(Y) + H(X|Y) = \tfrac{5}{2}$
(c) $I(X;Y) = 1 - H(X|Y)$
(d) $H(Y) \geq \tfrac 32$

Mostrar solución

La respuesta correcta es (a) ★

La afirmación $I(X;Y) = 1 - H(X|Y)$ es verdadera porque $H(X)=1$. $H(Y)\geq \tfrac 32$ también es verdadera porque el condicionamiento reduce la entropía. $H(Y) + H(X|Y) = H(X,Y) = H(X)+H(Y|X) = \tfrac 52$. La entropía conjunta no es $3$.
